## Getting started with Cursor

The CDS MCP server is included in the `@cbhq/cds-common` package which is installed by default for anyone using CDS.

Run `npx cds-mcp-setup` to setup the MCP server.

This installs the MCP server in `<workspace_root>/.cursor/mcp.json` and adds Cursor rules for CDS in `<workspace_root>/.cursor/rules/cds.mcp`.

:::tip
Make sure to enable the MCP server after installation. Go to `Cursor Settings` &rarr; `Tools & Integrations` &rarr; `MCP Tools` to check which MCP servers are installed.
:::

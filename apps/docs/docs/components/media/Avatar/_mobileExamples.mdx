### Avatar shapes

We offer three avatar shapes: Circle, Square, and Hexagon.

```jsx
<HStack gap={1}>
  <Avatar src={assets.btc.imageUrl} shape="circle" />
  <Avatar src={assets.btc.imageUrl} shape="square" />
  <Avatar src={assets.btc.imageUrl} shape="hexagon" />
</HStack>
```

### Avatar with fallback image

If you do not provide a src or if the provided src does not resolve, Avatar will surface a placeholder image.

```jsx
<VStack gap={2}>
  <HStack gap={2} alignItems="center">
    <Avatar alt="Sneezy" size="m" />
    <Avatar alt="Sneezy" size="m" selected />
    <Avatar alt="Happy" size="m" shape="square" />
    <Avatar alt="Happy" size="m" shape="square" selected />
  </HStack>
  <HStack gap={2} alignItems="center">
    <Avatar alt="Sneezy" />
    <Avatar alt="Sneezy" selected />
    <Avatar alt="Happy" shape="square" />
    <Avatar alt="Happy" shape="square" selected />
  </HStack>
  <HStack gap={2} alignItems="center">
    <Avatar alt="Sneezy" size="xl" />
    <Avatar alt="Sneezy" size="xl" selected />
    <Avatar alt="Happy" size="xl" shape="square" />
    <Avatar alt="Happy" size="xl" shape="square" selected />
  </HStack>
  <HStack gap={2} alignItems="center">
    <Avatar alt="Sneezy" size="xxl" />
    <Avatar alt="Sneezy" size="xxl" selected />
    <Avatar alt="Happy" size="xxl" shape="square" />
    <Avatar alt="Happy" size="xxl" shape="square" selected />
  </HStack>
</VStack>
```

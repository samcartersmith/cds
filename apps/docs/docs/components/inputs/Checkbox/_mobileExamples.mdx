Checkbox UI and state management are separated so that you can swap in any form or state code you like. CDS provides 2 hooks that pair nicely with Checkbox and CheckboxGroup for your convenience.

### Single Checkbox

```jsx
function CheckboxDemo() {
  const [isChecked, setIsChecked] = useState(false);
  return (
    <Checkbox value="hyped" onChange={setIsChecked} checked={isChecked}>
      Are you hyped about CDS?
    </Checkbox>
  );
}
```

If you don’t want to use the default label or layout style, you can just not provide `children` to the `Checkbox` component. However, you should think about wrapping the custom label and `Checkbox` in a `<label>` if it makes sense to increase the tap target of the component and allow users to toggle by pressing on the label as well.

```jsx
function CheckboxWithCustomLabelDemo() {
  const [isChecked, setIsChecked] = useState(false);
  return (
    <Box as="label" justifyContent="space-between" maxWidth="300px">
      <Text font="headline" as="span">
        I want special treatment.
      </Text>
      <Checkbox value="hyped" onChange={setIsChecked} checked={isChecked} />
    </Box>
  );
}
```

### CheckboxGroup

```jsx
function CheckboxGroupDemo() {
  // static values should live outside the component in product code.
  const options = {
    'fish-taco': 'Fish tacos',
    puttanesca: 'Spaghetti alla puttanesca',
    'hamachi-salad': 'Hamachi salad',
  };
  const optionValues = Object.keys(options);

  // example component using CheckboxGroup
  const [selectedValues, setSelectedValues] = useState(optionValues);
  return (
    <>
      <Text font="headline" as="h1" id="order-dinner-label">
        Order Dinner
      </Text>
      <CheckboxGroup
        aria-labelledby="order-dinner-label"
        name="checkbox-group"
        selectedValues={setSelectedValues}
        onChange={toggle}
      >
        {Object.entries(options).map(([value, label]) => (
          <Checkbox key={value} value={value}>
            {label}
          </Checkbox>
        ))}
      </CheckboxGroup>
    </>
  );
}
```

### Customizing Colors

Like other control components (i.e. Radio, Switch), you can customize the color of the Checkbox by setting the `controlColor` prop.

```jsx
function CustomColors() {
  const [checked, setChecked] = useState(false);

  return (
    <Checkbox
      controlColor="accentBoldGreen"
      onChange={() => setChecked((s) => !s)}
      checked={checked}
    >
      Custom control color
    </Checkbox>
  );
}
```

For more advanced color customization, you can use additional style props like `background`, `borderColor`, and `borderRadius`:

```jsx
function AdvancedCustomColors() {
  const [checked, setChecked] = useState(false);

  return (
    <Checkbox
      background={checked ? 'accentBoldPurple' : 'bgNegative'}
      borderColor={checked ? 'bgPositive' : 'bgPrimary'}
      borderRadius={800}
      checked={checked}
      controlColor="bgPositive"
      onChange={() => setChecked((s) => !s)}
    >
      Advanced styling
    </Checkbox>
  );
}
```

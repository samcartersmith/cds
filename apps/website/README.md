Full documentation for the go/cds website can be found in our [docs](https://github.cbhq.net/frontend/cds/blob/master/docs/website.md).

## Commands

| Script                                                 | Description                          |
| ------------------------------------------------------ | ------------------------------------ |
| `yarn nx run website:adoption`                         | Prepare adoption numbers             |
| `yarn nx run website:start`                            | Start website dev server             |
| `yarn nx run website:serve`                            | Serve website production build       |
| `yarn nx run website:build`                            | Build website                        |
| `yarn nx run website:format`                           | Format website code                  |
| `yarn nx run website:lint`                             | Lint website code                    |
| `yarn nx run website:lint-styles`                      | Lint website styles                  |
| `yarn nx run website:typecheck`                        | Typecheck website code               |
| `yarn nx run website:deprecations`                     | Update deprecations data             |
| `yarn nx run website:generateAdoptionAndImpactReports` | Generate adoption and impact reports |

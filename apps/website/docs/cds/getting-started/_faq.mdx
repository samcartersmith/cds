## FAQ

- Ensure that your html has `<meta charset="UTF-8">` which is required for the icon font in CDS web.

- To figure out what CDS primitive or component is being used in the design, you can inspect the target in Figma and look for component label, color palette name, or spacing size.

- example 1. Use `Headline` typography for the text.

- example 2. Use `postive` palette color for the background. 🌙 indicates that the palette is for dark mode. In most cases, light and dark mode can have the same styling.

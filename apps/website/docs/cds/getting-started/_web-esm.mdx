## Web ESM

If your web environment does not have a css loader for node_modules (nextjs, Remix etc.)

### Setup

<Npmrc />

### Install

<CodeBlock language="bash">{`yarn add @cbhq/cds-web@^${metadata.version}-esm.0`}</CodeBlock>

### Add root imports

```tsx
import '@cbhq/cds-web/assets/main.css'; // This will load all styles that are necessary for CDS.
import '@cbhq/cds-web/globalImports'; // This will load the focus-visible polyfill.
```

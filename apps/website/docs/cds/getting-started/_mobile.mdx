### Babel Plugins

Our cds-mobile package uses react-native-reanimated under the hood. This requires you to add [their babel plugin](https://docs.swmansion.com/react-native-reanimated/docs/2.x/fundamentals/installation/#babel-plugin) to your babel.config.js. **Note: Reanimated recommends putting their plugin at the end of your plugin list**

```tsx
  module.exports = {
    presets: [
      ...
    ],
    plugins: [
      ...
      'react-native-reanimated/plugin',
    ],
  };
```

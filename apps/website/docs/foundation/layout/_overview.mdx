## Anatomy

<DocImage category="foundation" component="layout" name="anatomy" format="png" />

1. **Sidebar**

   - Part of app frame
   - Fixed width of 240px when expanded, 88px when collapsed
   - Product teams can choose to set their sidebars to be expanded or collapsed but are advised to remember the user’s choice and not flip back and forth based on product

2. **Navigation Bar**

   - Part of app frame
   - Fluid

3. **Page**

   - Area in which sections sit in
   - Each section can be fixed or fluid

### Section

<DocImage category="foundation" component="layout" name="anatomy_section" format="png" />

A page is made up of sections. Sections are made up of these elements:

1. **Section**

   - Parent layout container without padding

2. **Border**

   - Border should usually be on the right and/or bottom

3. **Spacing**

   - Also known as padding
   - Currently has a value of spacing 3

4. **Content**

## Types

### Layouts

#### Primary and Secondary Layout

- Most common layout for consumer
- 2-column layout with primary and secondary widths
- Primary (⅔ of page)
- Secondary (⅓ of page)

### Fixed and Fluid

Depending on the product’s needs, sections that live on the page can be fixed or fluid. Fluid layouts should be reserved for dense layouts or single-column layouts with tables.

<DocImage category="foundation" component="layout" name="fixedfluid" format="png" />

**Mix and match**
On a single page, sections can be fixed or fluid. It’s okay to mix and match.

<DocImage category="foundation" component="layout" name="mixmatch1_do" format="png" />

### Dividers

Dividers go edge-to-edge to divide each section.

<DocImage category="foundation" component="layout" name="dividers" format="png" />

If there is a sidebar and sections have max values (versus fluid), the page should be left-aligned and a divider should be added to the far right to contain the page.

<DocImage category="foundation" component="layout" name="dividers_1920" format="png" />

If there is no sidebar and sections have max values, the page should be centered with divider lines on the left and right of the page.

<DocImage category="foundation" component="layout" name="dividers_center" format="png" />

### Length of text

Be mindful of the length of text in a fluid layout. Ideally there should be up to 40–60 characters per line or 13 English words per line. Anything more will negatively impact the design. The suggested max width for the body section is 800px.

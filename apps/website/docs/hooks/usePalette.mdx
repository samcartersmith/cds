> [@cbhq/cds-(web|mobile)/hooks/usePalette](https://github.cbhq.net/frontend/cds/blob/master/packages/web/hooks/usePalette.ts)

Returns the palette from the closest parent `<ThemeProvider />`.

- On web this will be CSS variables.
- On mobile this will be an object of rgba strings.

```jsx live
function Example() {
  const palette = usePalette();
  return (
    <VStack gap={1}>
      {Object.entries(palette).map(([key, value]) => (
        <HStack gap={1}>
          <TextLabel1 as="span">{`${key}:`}</TextLabel1>
          <TextLabel2 as="span">
            <code>{JSON.stringify(value)}</code>
          </TextLabel2>
        </HStack>
      ))}
    </VStack>
  );
}
```

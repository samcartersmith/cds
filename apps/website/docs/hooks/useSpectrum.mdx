> [@cbhq/cds-(web|mobile)/hooks/useSpectrum](https://github.cbhq.net/mono/repo/blob/master/eng/shared/design-system/web/hooks/useSpectrum.ts)

Returns the active spectrum (`light` or `dark`) from the closest parent `<ThemeProvider />`.

```jsx live
function Example() {
  const spectrum = useSpectrum();
  return (
    <ExampleWithThemeToggles>
      <TextDisplay1 as="h1">{spectrum}</TextDisplay1>
    </ExampleWithThemeToggles>
  );
}
```

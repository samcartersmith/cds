> [@cbhq/cds-common/overlays/useModal](https://github.cbhq.net/frontend/cds/blob/master/packages/common/overlays/useModal.ts)

Consumes `useOverlay()`. Check more info about it in [useOverlay](/hooks/useOverlay).

### Usage

```
const { openModal, closeModal } = useModal();

openModal(<Modal />);
closeModal();
```

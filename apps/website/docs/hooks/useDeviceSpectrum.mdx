> [@cbhq/cds-(web|mobile)/hooks/useDeviceSpectrum](https://github.cbhq.net/mono/repo/blob/master/eng/shared/design-system/web/hooks/useDeviceSpectrum.ts)

Returns the spectrum from the device

- On web this will be the result of the `(prefers-color-scheme: dark)` media query
- On mobile this will be the result of the react-native `useColorScheme` hook

The hook tracks preferences updates on the operating system and will auto update the app / web app.

```jsx live
function Example() {
  const spectrum = useDeviceSpectrum();
  return (
    <VStack gap={1}>
      <HStack gap={1}>
        <TextLabel1 as="span">{spectrum}</TextLabel1>
      </HStack>
    </VStack>
  );
}
```

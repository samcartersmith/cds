> [@cbhq/cds-common/visiualizations/useSparklinePath](https://github.cbhq.net/frontend/cds/blob/master/packages/common/visiualizations/useSparklinePath.ts)

Returns a svg path string.

```jsx live
function Example() {
  const data = [20, 30, 5, 45, 0];
  const dimensions = { width: 400, height: 200 };
  const path = useSparklinePath({ ...dimensions, data });
  return (
    <ExampleWithThemeToggles>
      <VStack gap={2}>
        <TextBody as="span">{JSON.stringify(path)}</TextBody>
        <Sparkline {...dimensions} path={path} color={assets.btc.color} />
      </VStack>
    </ExampleWithThemeToggles>
  );
}
```

## Params

| Name   | Value                  | Required |
| ------ | ---------------------- | -------- |
| width  | `number`               | true     |
| height | `number`               | true     |
| data   | `string[] or number[]` | true     |

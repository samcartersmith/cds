> [@cbhq/cds-(web/mobile)/overlays/useToast](https://github.cbhq.net/mono/repo/blob/master/eng/shared/design-system/web/overlays/useToast.ts)

Consumes `PortalProvider` and returns `show()`, `hide()`, and `clearQueue()` to control toast.

## Prerequisite

Component should be wrapped by `PortalProvider`.

## API

### `show(text, options)`

If no toast is showing, it will display a toast.

If a toast is showing, it will add a new toast to the queue that will be displayed later.

**Example usage:**

```jsx
const toast = useToast();

toast.show('Toast copy', {
  action: { label: 'Action', onPress: () => console.log('action pressed') },
  dangerouslySetDuration: 5000,
  hideCloseButton: true,
  onWillHide: () => {
    console.log('toast hiding');
  },
  onDidHide: () => {
    console.log('toast hidden');
  },
});
```

### `hide()`

Hide the current toast.

**Example usage:**

```jsx
const toast = useToast();

toast.hide();
```

### `clearQueue()`

Hide the current toast and clear queued toasts.

**Example usage:**

```jsx
const toast = useToast();

toast.clearQueue();
```

## [3.0.0](/changelog/mobile#300-10192022-0222-pm-pst)

### Gesture Handler Upgrade

Peer dependency `react-native-gesture-handler` was upgraded to version 2, which was a complete API rewrite and is _not_ backwards compatible. You will need to upgrade to version `2.6.2`.

### Sparkline Interactive Aliases

Replaced unicode values for SparklineInteractiveHeader subhead signs with [aliases](https://github.cbhq.net/frontend/cds/blob/master/packages/common/tokens/sparkline.ts#L21).

```diff
- sign: '–' | '+' | '' | '\u2197 ' | '\u2198 ';
+ sign: 'positive' | 'negative' | 'upwardTrend' | 'downwardTrend' | '';
```

# Migrations

Breaking releases of CDS core libraries can be found in this directory. We will always provide a migration guide to help your app move to the latest version.

We strongly recommend that you use `cds-migrator` to automatically migrate deprecations to their respective replacements. In the case that a migration is not possible, we will provide a manual migration guide in the console. Refer to your specific release version in the sidebar for more information.

:::warning
Please only use v2.1.0 and higher as v2.0.0 is incompatible with v5.0.0 and higher of web and mobile packages.
:::

This release introduces breaking changes to some illustration dimensions and names. You can use the `cds-migrator` to update all instances, however since dimensions were altered this will require manual review, since it could cause visual layout shift.

You can optionally use the `scaleMultiplier` prop to scale the illustration back to the original size, however please note this goes against most recent design guidance.

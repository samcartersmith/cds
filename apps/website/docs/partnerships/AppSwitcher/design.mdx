## Behavior

<br />

### Opening and closing

<br />

<ul>
  <li>When the App Switcher Icon Button is triggered, a background overlay appears.</li>
  <li>App Switcher animates in from the top of the screen.</li>
  <li>Selecting a product or clicking outside of the Menu will close the App Switcher.</li>
</ul>

<br />

### Scrolling

<br />

<ul>
  <li>A scrollbar will appear in the Menu when the height of the Popover Menu exceeds 600px.</li>
</ul>

### Overflow

<br />

<ul>
  <li>
    If a Product Name is longer than the width of the Square Button container, the name will
    truncate to one line line with an ellipses.
  </li>
  <li>
    On hover, the full name will show on two lines. This should not push down or disrupt the other
    items in the Menu.{' '}
  </li>
</ul>

This page demonstrates the recipes that CDS recommends consumers adopt in order to build out an AppSwitcher component.

These Recipes are intentionally NOT API-driven and simply serve as guidance on how to implement an AppSwitcher. It is a product team's responsibility to turn these recipes into reusable components.

<br />

We have also added demo data to show a more realistic example; note that this is for demo purposes only, we anticipate the consumer will need to modify the content component to work with Relay/GraphQL.

<br />

## CDS Components

<br />

The App Switcher recipe is composed of the following CDS components:

- [Dropdown](/components/dropdown)
- [NavigationIconButton](/components/navigation-icon-button)
- [TileButton](/components/tile-button)
- [SectionTitle](/components/section-title)

<br />

Some key details about the abstractions in this recipe:

<br />

Dropdown

- Dropdown contains the prescriptive layout/overlay logic shared between App and UserSwitcher.

<br />

AppSwitcherContent

- This is a component that was built simply for demonstration purposes.
- We technically consider this component more of a recipe and is demo'd in the second example.
- CDS will not expose a component like this, however we expect Identity will implement a product component very similar to this.

<br />

## AppSwitcherContent Recipe

<br />

Above we encapsulated the content code into a component to cleanly demonstrate the high level architecture of the AppSwitcher Recipe.

The below example is meant to expose the AppSwitcherContent Recipe because we have no intention of exposing the AppSwitcherContent component.
`AppSwitcherContent` was simply built out for demonstration purposes and we expect it will be different when consumers implement it with GraphQL etc.

<br />

```jsx live
function AppSwitcherContentRecipeExample() {
  function NoopFn() {
    console.log('pressed');
  }

  const appSwitcherData = {
    sections: [
      {
        sectionTitle: 'For Individuals',
        tiles: [
          { pictogram: 'cardNavigation', title: 'Card', onPress: NoopFn },
          {
            pictogram: 'earnNavigation',
            title: 'Earn',
            onPress: NoopFn,
          },
          { pictogram: 'walletNavigation', title: 'Wallet', onPress: NoopFn, count: 9 },
          { pictogram: 'proNavigation', title: 'Pro', onPress: NoopFn },
          {
            pictogram: 'privateClientNavigation',
            title: 'Private Client',
            onPress: NoopFn,
            count: 4,
          },
        ],
      },
      {
        sectionTitle: 'For Businesses',
        tiles: [
          { pictogram: 'primeNavigation', title: 'Prime', onPress: NoopFn, count: 10 },
          { pictogram: 'commerceNavigation', title: 'Commerce', onPress: NoopFn },
          { pictogram: 'exchangeNavigation', title: 'Exchange', onPress: NoopFn },

          { pictogram: 'assetHubNavigation', title: 'Asset Hub', onPress: NoopFn, count: 2 },
          { pictogram: 'custodyNavigation', title: 'Custody', onPress: NoopFn, count: 999 },
          { pictogram: 'analyticsNavigation', title: 'Analytics', onPress: NoopFn },

          { pictogram: 'venturesNavigation', title: 'Ventures', onPress: NoopFn },
          { pictogram: 'institutionalNavigation', title: 'Institutional', onPress: NoopFn },
        ],
      },
    ],
  };

  function AppSwitcherContentSection({ columns, data }) {
    const rows = Math.ceil(data.tiles.length / columns);
    return (
      <>
        <VStack spacingHorizontal={2}>
          <SectionTitle text={data.sectionTitle} />
          {Array.from(Array(rows)).map((item, row) => {
            return (
              <HStack gap={0.5}>
                {data.tiles
                  // We are able to infer the interval because we know how big dataset is + what row we are on.
                  .slice(row * columns, row * columns + columns)
                  .map((props) => {
                    return (
                      <HStack zIndex={getZIndexFromRow(row, rows)} gap={0.5} spacingBottom={0.5}>
                        <TileButton {...props} />
                      </HStack>
                    );
                  })}
              </HStack>
            );
          })}
        </VStack>
      </>
    );
  }

  function AppSwitcherContentRecipe({ columns = 3, data = appSwitcherData }) {
    return (
      <VStack>
        {data.sections.map((section, idx) => {
          return (
            <>
              <AppSwitcherContentSection data={section} columns={columns} />
              {data.sections.length - 1 !== idx && <Divider spacingVertical={1} />}
            </>
          );
        })}
      </VStack>
    );
  }

  return (
    <VStack justifyContent="center" spacing={4}>
      <VStack maxWidth={359}>
        <AppSwitcherContentRecipe />
      </VStack>
    </VStack>
  );
}
```

# Deprecations

Deprecations will be deleted during semi-annual breaking releases. Use the sidebar to navigate to each major release's deprecations. To learn more about the deprecation process please refer to our [deprecation docs](https://github.cbhq.net/frontend/cds/blob/master/docs/conventions/deprecations.md).

We strongly recommend that you use `cds-migrator` to automatically migrate deprecations to their respective replacements. In the case that a migration is not possible, we will provide a manual migration guide in the console. Refer to the [Migration Guide](../../guides/migration) for the appropriate breaking release for more information.

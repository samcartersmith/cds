This is a list of changes in CDS v6.0.0. Please use the <code>cds-migrator</code> package and respective [migration guide](/guides/migration/web-and-mobile/6-0-0/) to update your codebase.

The Sparkline components have moved to the <code>@cbhq/cds-web-visualization</code> and <code>@cbhq/cds-mobile-visualization</code> packages.

The <code>@cbhq/cds-web-overlay</code> package is deprecated and all components should be imported from <code>@cbhq/cds-web</code> instead.

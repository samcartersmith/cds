Icons are hand-crafted symbols for common visual actions and items.

For an icon to render, it requires a unique name that maps to an upstream asset in Figma (this will autocomplete when using TypeScript), and a target size (xs, s, m, l).

```jsx
<Icon name="caretRight" size="m" />
```

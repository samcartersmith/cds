By default, icons are considered decorative, and are hidden from assistive technology. When used within a component that has no label (like an icon button), a label should be provided to the parent component using `accessibilityLabel`.

```jsx
<IconButton name="calendar" accessibilityLabel="Recurring calendar" />
```

However, if the icon is used stand-alone, the icon can be labeled through another element's content by mapping to their ID.

```jsx
<Icon name="calendar" labeledBy="recurring-calendar" />

<div id="recurring-calendar">The recurring calendar for your BTC transactions.</div>
```

Or by providing its own `title` alongside a custom label ID.

```jsx
<Icon name="calendar" title="Recurring calendar" labeledBy="recurring-calendar" />
```

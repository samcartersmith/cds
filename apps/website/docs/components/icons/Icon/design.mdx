Iconography is a key piece of our user interface. It can be leveraged both as a wayfinding tool and also as a contextual piece of visual information.

We use it to guide users through various experiences, encourage people to take action, and allow information to be parsed more effectively and efficiently.

We index highly on universal symbols over metaphorical abstraction while utilizing a style that reflects our brand personality though it’s heavy weight, geometric base and crisp angularity.

# Size

Icons are scale aware and render at different sizes depending on the active scale.
Since an icon can render at one size in normal scale and a different size in dense scale, we use t-shirt naming (xs-l) to represent the 4 sizes we offer.

# Color

All of our icons are designed using a single color. This is done to achieve maximum flexibility of usage and can be edited to suit the needs of a particular experience or flow.

While the color can be edited the most common use cases are default blue, inactive gray and black.

### labeledBy <Badge variant="web" />

If the icon is labelled by an independent element, then provide that elements id. This is used to set the aria-labelledby value on the icon. If the `title` prop is also provided, then `labeledBy` will be used to set the id on the `<title>` element.

<ApiDetails
  componentName="NavigationIcon"
  propertyName="labeledBy"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['string']}
/>

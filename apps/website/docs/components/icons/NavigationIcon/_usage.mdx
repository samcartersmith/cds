For a navigation icon to render, it requires a unique name that maps to an upstream asset in Figma (this will autocomplete when using TypeScript), and a target size (s, m, l).
We added an active prop to toggle the state of the Icon.

```jsx live
<VStack gap={3}>
  <NavigationIcon name="bell" size="m" active />
  <NavigationIcon name="bell" size="m" active={false} />
</VStack>
```

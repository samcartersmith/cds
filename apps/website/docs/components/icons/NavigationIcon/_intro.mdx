Navigation icons are custom icons created for the Navigation component at Coinbase. We **DO NOT** recommend using these icons for outside of Navigation. Recommended size for Navigation Icons is 24px (M). If you want to contribute an icon, please make a request at <a href="https://jira.coinbase-corp.com/secure/CreateIssueDetails!init.jspa?pid=15203&issuetype=3&priority=4&labels=Illustration&labels=Icon&labels=Request&components=20226&assignee=dominic.flask@coinbase.com&summary=%5BIllustration%20or%20Icon%20Request%5D%20&description=(_Please%20make%20sure%20you%E2%80%99re%20requesting%20support%20at%20least%202%20weeks%20prior%20to%20the%20due%20date.%20%20Please%20be%20as%20descriptive%20as%20possible,%20this%20will%20set%20up%20the%20Illustration%20Team%20for%20success._)%20%0A%0A*Estimated%20Scope:*%20(_Ideal%20delivery%20date?%20Any%20important%20dates%20we%20should%20know?_)%0A%0A*Project%20Description:*%20(_Provide%20brief,%20examples,%20and%20DRI%E2%80%99s_)%0A%0A*Resources:*%20(_Provide%20links,%20Figma,%20decks,%20documentation_)">go/icon-request</a>

We offer an active and inactive state for these icons. When the navigation icon is active, it is meant to indicate that the icon is pressed or a consumer is on a particular screen. When the navigation icon is inactive, it means no activity is being triggered on this icon.

You now have the ability to search for an icon using keywords recommended by CDS Illustrators. For example, if you search "average", it will recommend trading. Our designers believe that if you want an icon to represent average, trading would be a good icon candidate.

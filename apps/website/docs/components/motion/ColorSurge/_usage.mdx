### How to

Place the `ColorSurge` component inside the container that you want to have color surge.
CDS components using ColorSurge: [Toast](/components/toast), [TextInput](/components/text-input/?page=implementation#enableColorSurge)

**A few gotchas:**

- Make sure the component is placed before other Interactables to avoid overlapping.
- You may need to add `overflow:hidden` to the parent container if it has border radius.
- You may need to add `position:relative` or `position:absolute` to the parent container to make `ColorSurge` fill.

```jsx
<ContainerBox borderRadius="rounded" overflow="hidden" position="relative">
  <ColorSurge />
  <Component />
</ContainerBox>
```

### Animate on mount

By default, the animation plays once on component mount.
You can disable this behaviour using the `disableAnimateOnMount` prop and trigger the animation using the [imperative method](#imperative-trigger).

```jsx live
<VStack gap={3}>
  <Box spacing={3} borderRadius="rounded" overflow="hidden" position="relative" bordered>
    <ColorSurge />
    <TextBody as="p">Default</TextBody>
  </Box>
  <Box spacing={3} borderRadius="rounded" overflow="hidden" position="relative" bordered>
    <ColorSurge background="positive" />
    <TextBody as="p">Positive</TextBody>
  </Box>
  <Box spacing={3} borderRadius="rounded" overflow="hidden" position="relative" bordered>
    <ColorSurge background="negative" />
    <TextBody as="p">Negative</TextBody>
  </Box>
</VStack>
```

### Imperative Trigger

You can use the `play` method to imperatively trigger the animation,
it also accepts background color as the argument.

- If no background is specificed as argument,
  it will use the background color from the `background` prop.
- If no `background` prop is specificed, it will use the default background color which is `primary`.

```jsx live
function BasicColorSurge() {
  const ref = useRef();

  const handleSurgePrimary = useCallback(() => ref.current.play('primary'), []);
  const handleSurgePositive = useCallback(() => ref.current.play('positive'), []);
  const handleSurgeNegative = useCallback(() => ref.current.play('negative'), []);

  return (
    <VStack gap={3}>
      <Box spacing={3} borderRadius="rounded" overflow="hidden" position="relative" bordered>
        <ColorSurge ref={ref} disableAnimateOnMount />
        <TextBody as="p">Default</TextBody>
      </Box>
      <HStack gap={3}>
        <Button onPress={handleSurgePrimary}>Surge - Primary</Button>
        <Button onPress={handleSurgePositive}>Surge - Positive</Button>
        <Button onPress={handleSurgeNegative}>Surge - Negative</Button>
      </HStack>
    </VStack>
  );
}
```

### Usage Examples

#### TextInput

```jsx live
<VStack gap={3}>
  <TextInput label="Default" placeholder="placeholder" variant="foregroundMuted" enableColorSurge />
  <TextInput label="Positive" placeholder="placeholder" variant="positive" enableColorSurge />
  <TextInput label="Negative" placeholder="placeholder" variant="negative" enableColorSurge />
</VStack>
```

#### Toast

```jsx live
function BasicToast() {
  const toast = useToast();

  const handleShowToast = useCallback(() => {
    toast.show('Copied to clipboard');
  }, [toast]);

  return <Button onPress={handleShowToast}>Show Toast</Button>;
}
```

Please see [here](/recipes/sparkline-interactive) for sample code with SparklineInteractiveHeader.

#### Default usage

```jsx live
<Box spacing={3} width="100%">
  <SparklineInteractivePriceWithHeader data={sparklineInteractiveData} strokeColor="#F7931A" />
</Box>
```

#### fill

The fill will be added by default

```jsx live
<Box spacing={3} width="100%">
  <SparklineInteractivePriceWithHeader data={sparklineInteractiveData} fill strokeColor="#F7931A" />
</Box>
```

#### Custom label

```jsx live
<Box spacing={3} width="100%">
  <SparklineInteractivePriceWithHeader
    data={sparklineInteractiveData}
    strokeColor="#F7931A"
    labelNode={
      <HStack gap={1} alignItems="center" spacingBottom={0}>
        <Icon name="wallet" size="s" />
        <TextTitle3 as="span">CustomHeader</TextTitle3>
      </HStack>
    }
  />
</Box>
```

#### Compact

```jsx live
<Box spacing={3} width="100%">
  <SparklineInteractivePriceWithHeader
    compact
    fill
    strokeColor="#F7931A"
    data={sparklineInteractiveData}
  />
</Box>
```

## Props

### background <Badge variant="danger" order={1} />

Only use if you have a non CDS color powering the background color that a Sparkline will appear on top of. This is needed to validate the the `color` prop is an accessible color, otherwise it will pick an appropriate replacement. Accepts any valid color (hex, rgb, rgba).

<ApiDetails
  componentName="SparklineGradient"
  propertyName="background"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

### path

Svg path as string. CDS offers a `useSparklinePath` which is useful to generate this string. This is accessible via `import { useSparklinePath } from @cbhq/cds-common/visualizations/useSparklinePath;`. Alternatively, you can use product tailored tooling to generate the SVG path. This component only requires a valid path string is provided.

<ApiDetails
  componentName="SparklineGradient"
  propertyName="path"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

### yAxisScalingFactor

Scales the sparkline to show more or less variance. Use a number less than 1 for less variance and a number greater than 1 for more variance. If you use a number greater than 1 it may clip the boundaries of the sparkline.

  componentName="SparklineGradient"
  propertyName="yAxisScalingFactor"
  required={false}
  defaultValue=""
  mobileOptions={['number']}
  webOptions={['number']}
/>

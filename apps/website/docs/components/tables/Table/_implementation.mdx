import { ApiDetails } from ':cds-website/components/ApiDetails';
import { Badge } from ':cds-website/components/Badge';

## Props

### bordered <Badge variant="web" />

When provided, well apply a bordered around the entire table

<ApiDetails
  componentName="Table"
  propertyName="bordered"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['false', 'true']}
/>

### cellSpacing <Badge variant="web" />

Provide custom cell spacing for all child TableCells

<ApiDetails
  componentName="Table"
  propertyName="cellSpacing"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['TableCellSpacing']}
/>

### children <Badge variant="web" />

Children are required, and should always include a TableBody. TableHeader and TableFooter are both optional, and will magically flow into the correct place in the table (top/bottom)

<ApiDetails
  componentName="Table"
  propertyName="children"
  required={true}
  defaultValue=""
  mobileOptions={[]}
  webOptions={[
    'Iterable<ReactNode>',
    'ReactElement<any, string | JSXElementConstructor<any>>',
    'ReactPortal',
    'false',
    'null',
    'number',
    'string',
    'true',
    '{}',
  ]}
/>

### tableLayout <Badge variant="web" />

Use tableLayout=fixed if you need full control over cell width

<ApiDetails
  componentName="Table"
  propertyName="tableLayout"
  required={false}
  defaultValue="'auto'"
  mobileOptions={[]}
  webOptions={['auto', 'fixed']}
/>

### testID <Badge variant="web" />

Used to locate this element in unit and end-to-end tests. Under the hood, testID translates to data-testid on Web. On Mobile, testID stays the same - testID

<ApiDetails
  componentName="Table"
  propertyName="testID"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['string']}
/>

### variant <Badge variant="web" />

The variant prop allows clients to use a CDS approved style for their Table.

<ApiDetails
  componentName="Table"
  propertyName="variant"
  required={false}
  defaultValue="undefined"
  mobileOptions={[]}
  webOptions={['default', 'graph', 'ruled']}
/>

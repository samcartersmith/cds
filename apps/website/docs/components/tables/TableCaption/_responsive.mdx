## Responsive Styles

You can create responsive layouts by passing a configuration object to a `TableCaption` and specify unique styles for phones, tablets, or desktops.

Please note, there are no default responsive styles for `TableCaption`. This is an additive feature that you will need to configure yourself.

### Supported Properties

- innerSpacing
- outerSpacing

### Usage

Always hoist or memoize your responsive style configurations.

```jsx live
function ResponsiveCaption() {
  const responsiveConfig = useMemo(
    () => ({
      phone: {
        innerSpacing: {
          spacingHorizontal: 2,
        },
        outerSpacing: {
          spacingVertical: 1,
        },
      },
      desktop: {
        innerSpacing: {
          spacingHorizontal: 4,
        },
        outerSpacing: {
          spacingVertical: 2,
        },
      },
    }),
    [],
  );

  return (
    <Table bordered>
      <TableCaption responsiveConfig={responsiveConfig}>
        TableCaption with Responsive Configuration
      </TableCaption>
    </Table>
  );
}
```

## Responsive Styles

**Web only** You can create responsive layouts by passing a configuration object to a TableCell and specify unique styles for phones, tablets, or desktops.

Please note, there are no default responsive styles for TableCell. This is an additive feature that you will need to configure yourself.

### Supported Properties

- Spacing
- Offset spacing
- Gap
- Flex styles (justify content, align content, align items, align self, flex direction, and flex wrap)
- Display
- Visibility

### Usage

Always hoist or memoize your responsive style configurations.

```jsx live
function ResponsiveTableCell() {
  const responsiveStylesConfig = useMemo(
    () => ({
      phone: {
        innerSpacing: {
          spacingHorizontal: 2,
        },
        outerSpacing: {
          spacingVertical: 1,
        },
      },
      desktop: {
        innerSpacing: {
          spacingHorizontal: 4,
        },
        outerSpacing: {
          spacingVertical: 2,
        },
      },
    }),
    [],
  );

  return (
    <Table>
      <TableBody>
        <TableRow>
          <TableCell responsiveStyles={responsiveStylesConfig} direction="horizontal">
            <TextHeadline as="h2">TableCell with responsive spacing</TextHeadline>
            <Button variant="secondary" compact onPress={() => console.log('Pressed')}>
              Export
            </Button>
          </TableCell>
        </TableRow>
      </TableBody>
    </Table>
  );
}
```

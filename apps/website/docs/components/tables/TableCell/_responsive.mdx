## Responsive Styles

You can create responsive layouts by passing a configuration object to a `TableCell` and specify unique styles for phones, tablets, or desktops.

Please note, there are no default responsive styles for `TableCell`. This is an additive feature that you will need to configure yourself.

### Supported Properties

- innerPadding
- outerPadding

### Usage

Always hoist or memoize your responsive style configurations.

```jsx live
function ResponsiveTableCell() {
  const responsiveConfig = useMemo(
    () => ({
      phone: {
        innerPadding: {
          spacingHorizontal: 2,
        },
        outerPadding: {
          spacingVertical: 1,
        },
      },
      desktop: {
        innerPadding: {
          spacingHorizontal: 4,
        },
        outerPadding: {
          spacingVertical: 2,
        },
      },
    }),
    [],
  );

  return (
    <Table>
      <TableBody>
        <TableRow>
          <TableCell responsiveConfig={responsiveConfig} direction="horizontal">
            <TextHeadline as="h2">TableCell with responsive spacing</TextHeadline>
            <Button variant="secondary" compact onPress={() => console.log('Pressed')}>
              Export
            </Button>
          </TableCell>
        </TableRow>
      </TableBody>
    </Table>
  );
}
```

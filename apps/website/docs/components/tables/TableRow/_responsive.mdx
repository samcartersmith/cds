## Responsive Styles

**Web only** You can create responsive layouts by passing a configuration object to a TableRow and specify unique styles for phones, tablets, or desktops. Responsive styles are only supported when a TableRow is `fullWidth`.

Please note, there are no default responsive styles for TableRow. This is an additive feature that you will need to configure yourself.

### Supported Properties

- innerSpacing
- outerSpacing

### Usage

Always hoist or memoize your responsive style configurations.

```jsx live
function ResponsiveTableRow() {
  const responsiveConfig = useMemo(
    () => ({
      phone: {
        innerSpacing: {
          spacingHorizontal: 2,
        },
        outerSpacing: {
          spacingVertical: 1,
        },
      },
      desktop: {
        innerSpacing: {
          spacingHorizontal: 4,
        },
        outerSpacing: {
          spacingVertical: 2,
        },
      },
    }),
    [],
  );

  return (
    <Table>
      <TableBody>
        <TableRow responsiveConfig={responsiveConfig} fullWidth>
          <TableCell
            direction="horizontal"
            title="TableRow with responsive spacing"
            end={
              <Button variant="secondary" compact onPress={() => console.log('Pressed')}>
                Export
              </Button>
            }
          />
        </TableRow>
      </TableBody>
    </Table>
  );
}
```

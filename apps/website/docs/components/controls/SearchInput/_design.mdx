  </ul>

## Cross-Platform

The main difference between Search on Mobile vs. Web is exiting the Search input.

On Web, users can simply click out or press tab to exit search. On Mobile, the Search icon will animate to a backArrow icon. Clicking on the backArrow icon will exit the search page.

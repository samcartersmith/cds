### inputNode <Badge variant="web" /> <Badge variant="danger" order={2} />

Customize the element which the input area will be rendered as. Adds ability to render the input area as a `<textarea />`, `<input />` etc... By default, the input area will be rendered as an `<input />`. Our default input handles all of the UI/Accessibility needs for your out of the box, but inputNode will not include those.
Use this at your own risk, and don't use unless ABSOLUTELY NECESSARY. You may see weird UI when focusing etc..

<ApiDetails
  componentName="TextInput"
  propertyName="inputNode"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['ReactElement<any, string | JSXElementConstructor<any>>']}
/>

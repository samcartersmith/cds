NativeTextArea is not a standalone component. The component is designed 
to be used in conjunction with TextInput to create multiline TextInput, also known as TextArea. 

NativeTextArea adds all the necessary CSS to the native `<textarea />`, so that it will render correctly inside the TextInput. Any API available in the [textarea MDN Doc](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/textarea) 
is also available in NativeTextArea

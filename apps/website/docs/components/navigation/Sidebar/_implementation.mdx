## Props

### accessibilityDescriptionId <Badge variant="web" />

Used to associate descriptive text to an input for accessibility

<ApiDetails
  componentName="Sidebar"
  propertyName="accessibilityDescriptionId"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['string']}
/>

### accessibilityHint <Badge variant="web" />

Accessibility hint for assisted devices when focused. Message should indicate the purpose of the control. If not supplied, will default to label

<ApiDetails
  componentName="Sidebar"
  propertyName="accessibilityHint"

  defaultValue=""
  mobileOptions={[]}
  webOptions={['string']}
/>

## Intro

The Sidebar is our primary navigation component. It provides access to the main features of the product, while making better use of the available space of larger displays.

## Usage

### Do

- Use the Sidebar to navigate the top level product hierarchy. This likely maps to the main and most commonly used product features.
- The Sidebar should be visible and accessible at all times.

### Don’t

- Show varying Sidebars within a single product. The Sidebar represents the primary navigation and should be consistent across your product. If you’re looking for a sub-navigation solution, please use Tabs in addition to keeping the Sidebar intact.

## Anatomy

The Sidebar supports an unlimited number of items and will be scrollable if the list expands beyond the visible area. We generally recommend keeping the primary navigation as simple as possible and limited to the most important features though.

### Extended & Collapsed Size

The Sidebar comes in two sizes: _Extended_ and _Collapsed_. By default, we will display the extended variant if the screen or window size is larger than 768px, and the collapsed variant for smaller screen or window sizes below that size.

<Image
  srcLight="/img/components/Sidebar/collapsed_light.jpg"
  srcDark="/img/components/Sidebar/collapsed_dark.jpg"
/>

In rare cases in which you’re dealing with extraordinarily complex or dense layouts, you can decide to overwrite this behavior and force the condensed variant on a particular screen, e.g. if your product feature includes a particular complex or dense user interface (e.g. advanced trading features). Please only do this if absolutely necessary to keep the general behavior as consistent as possible.

### Badging (CDS component coming soon)

The Sidebar will support badging for any item in the bar.

## Iconography

Icons in our Tab Bar component are part of a special navigation icon set.

> If you require a new icon, please create a ticket via [go/illustration-request](http://go/illustration-request).

## Content

Sidebar titles should be as concise as possible. Aim for 1–2 short words that match the Navigation Bar title.

Put the highest priority items at the top of the list.

You can use a Label to promote a sidebar item. Use only 1 label on a sidebar.

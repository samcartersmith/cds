## Props

### accessibilityHint

Accessibility hint for assisted devices when focused. Message should indicate the purpose of the control. If not supplied, will default to label

<ApiDetails
  componentName="DotSymbol"
  propertyName="accessibilityHint"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

### accessibilityLabel

Accessibility label describing the control. If no accessibilityLabel, it will set it to label Maps to `aria-label`

<ApiDetails
  componentName="DotSymbol"
  propertyName="accessibilityLabel"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

Accessibility prop that describes the component that labels it by its `id`

**https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/ARIA_Techniques/Using_the_aria-labelledby_attribute**

  componentName="DotSymbol"
  propertyName="accessibilityLabelledBy"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

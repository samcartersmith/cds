`PopoverTriggerWrapper` should only be used as a direct descendent of a `PopoverMenu`. This component should only be used when the top level node of a `PopoverTrigger` is not an interactable; like a layout component or a decorator like a `DotColor`.

### How to use it

You will need to wrap the entire trigger node in `PopoverTriggerWrapper` and then wrap the interactable part with `PopoverTrigger`. For example, say we want to have an `AvatarButton` as the trigger, but we also want to surface a `DotStatus`. This is how you would compose it:

```jsx
const Example = () => (
  <PopoverMenu>
    <PopoverTriggerWrapper>
      <DotStatusColor variant="positive" pin="top-start">
        <PopoverTrigger>
          <AvatarButton alt="Sneezy" />
        </PopoverTrigger>
      </DotStatusColor>
    </PopoverTriggerWrapper>
    {/* Popover Menu children */}
  </PopoverMenu>
);
```

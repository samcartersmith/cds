`Modal` is a component that disables the main window (but keeps it visible) and presents the user with a window that they must interact with before they can return to the parent application. This avoids interrupting the workflow on the main window and presents temporary information that’s separate from the user’s current context, while requiring an explicit action to exit. Modals help people to focus on self-contained tasks or a set of closely related options to ensure that people receive and, if necessary, act on critical information.

## Anatomy

<span>&nbsp;</span>

`Modals` are containers comprised of: Modal Header, Modal Body, and Modal Footer with a Background Overlay. Each section relates to the subject/topic of the Card. 

Modal Header provides high level information about the modal such as the simple task the user needs to complete (example: Borrow cash). 
Modal Body should be short and simple. Limit the number of inputs that the user needs to fill out since modals are, by nature, easy to dismiss and cancel out of.
Modal Footer is reserved for actions.

## Responsive

`Modal` responds to device and browser. 

<Image
  light="/img/components/overlays/modal/specs_light.png"
  dark="/img/components/overlays/modal/specs_dark.png"
/>

<span>&nbsp;</span>

When device width is more than 660px:
- **Buttons:** responsive and side-by-side
- **Maximum Width:** 612px
- **Minimum Width:** device-width with 24px margins from the left and right of the screen
- **Height:** defined by content
- **Maximum Height:** device-height of spacing `10` starting from the top of the screen
- **Style:** Rounded corners, `elevation, level 2`

<span>&nbsp;</span>

<Image
  light="/img/components/overlays/modal/fullscreen_light.png"
  dark="/img/components/overlays/modal/fullscreen_dark.png"
/>

<span>&nbsp;</span>

When device width is less than 414px:
- **Buttons:** block and stack
- **Width:** if device width is less than 414px, modal is 100% device-width with no margins
- **Height:** modal is 100% device-heigh with no margins
- **Style:** no rounded corners, no elevation

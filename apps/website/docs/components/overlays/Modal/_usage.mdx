### Get Started

1. Add [PortalProvider](/components/portal-provider) to the root of your app. Import it from `import { PortalProvider } from '@cbhq/cds-web/overlays/PortalProvider';`.
2. Compose the `Modal` with [ModalHeader](/components/modal-header), [ModalBody](/components/modal-body), and [ModalFooter](/components/modal-footer) for consistency:

```
<Modal>
	<ModalHeader />
	<ModalBody>
		<AnyContentYouWant />
	</ModalBody>
	<ModalFooter />
</Modal>
```

3. Wire up `visible` and `onRequestClose` to control visibility (see code examples below).
4. Include one or more of the following methods to exit the modal:

   - a close button in the header (present by default, can be hidden by setting `hideCloseButton` prop to `true`)
   - a cancel or dismiss button in `ModalFooter` (setup required, see code examples below)

   **Web only**

   - clicking outside of the modal (default behavior, can opt out by setting `disableOverlayPress` prop to `true`)
   - pressing the Esc key (default behavior, can opt out by setting `shouldCloseOnEscPress` prop to `false` but not recommended as it's bad for accessibility)

### Default Modal

`Modal` uses a React [Portal](https://reactjs.org/docs/portals.html) by default, which will port the node to the end of the `body` tag. You can set `disablePortal` to opt out of this behavior.

```jsx live
// import { PortalProvider } from '@cbhq/cds-web/overlays/PortalProvider';

function DefaultModalExample() {
  const [visible, { toggleOn, toggleOff }] = useToggler();
  const [scrolling, { toggle: toggleScrolling }] = useToggler();
  const triggerRef = useRef(null); // only required for web

  const handleDidClose = useCallback(() => {
    if (triggerRef.current) {
      triggerRef.current.focus(); // only required for web
    }
  }, []);

  return (
    <ExampleWithThemeToggles showFrontier>
      <PortalProvider>
        <ButtonGroup>
          <Button onPress={toggleOn} ref={triggerRef}>
            Open Modal
          </Button>
          <Switch onChange={toggleScrolling} checked={scrolling}>
            Show long content
          </Switch>
        </ButtonGroup>
        <Modal visible={visible} onRequestClose={toggleOff} onDidClose={handleDidClose}>
          <ModalHeader
            title="Default Modal"
            onBackButtonPress={toggleOff}
            backAccessibilityLabel="Back"
            closeAccessibilityLabel="Close"
          />
          <ModalBody>
            <TextBody as="p">{loremIpsum.repeat(scrolling ? 50 : 1)}</TextBody>
          </ModalBody>
          <ModalFooter
            primaryAction={<Button onPress={toggleOff}>Save</Button>}
            secondaryAction={
              <Button variant="secondary" onPress={toggleOff}>
                Cancel
              </Button>
            }
          />
        </Modal>
      </PortalProvider>
    </ExampleWithThemeToggles>
  );
}
```

### Programmatically Controlled Modal

`PortalProvider` is a convenient way for adding anything to the CDS portal. The CDS portal is what CDS uses to centralize all elements that need a portal and handles the zIndex of those pieces.
You can control `Modal` visibility programmatically using `openModal()` and `closeModal()` from the `useModal()` hook which consumes `PortalProvider` under the hood.

To **open** the modal, pass your `Modal` component into `openModal()` and pass in the `closeModal` handler to the `onRequestClose` prop to make the modal close properly on pressing the X button.

To **close** the modal, use the `closeModal()` function from `useModal()`.

Multiple modals? No problem, check out this example [here](/hooks/useOverlay#multiple).

```jsx live
// import { PortalProvider } from '@cbhq/cds-web/overlays/PortalProvider';
// import { useModal } from '@cbhq/cds-web/overlays/useModal';

function ControlledModalExample() {
  function ControlledModal() {
    const { openModal, closeModal } = useModal();
    const triggerRef = useRef(null); // only required for web

    const handleDidClose = useCallback(() => {
      if (triggerRef.current) {
        triggerRef.current.focus(); // only required for web
      }
    }, []);

    const handlePress = useCallback(
      () =>
        openModal(
          <Modal visible onRequestClose={closeModal} onDidClose={handleDidClose}>
            <ModalHeader
              title="Controlled Modal"
              onBackButtonPress={closeModal}
              backAccessibilityLabel="Back"
              closeAccessibilityLabel="Close"
            />
            <ModalBody>
              <TextBody as="p">{loremIpsum}</TextBody>
            </ModalBody>
            <ModalFooter primaryAction={<Button onPress={closeModal}>Save</Button>} />
          </Modal>,
        ),
      [openModal, closeModal],
    );

    return (
      <Box>
        <Button onPress={handlePress} ref={triggerRef}>
          Open Modal
        </Button>
      </Box>
    );
  }

  return (
    <PortalProvider>
      <ExampleWithThemeToggles>
        <ControlledModal />
      </ExampleWithThemeToggles>
    </PortalProvider>
  );
}
```

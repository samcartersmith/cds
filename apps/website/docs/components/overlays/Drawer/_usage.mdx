A `Drawer` is an elevated container that is pinned to one side of the screen. It also triggers an overlay. Its visibility is toggled by another component, typically a `Button` or a control, like a `Select`. It is the mobile equivalent of a `PopoverMenu`.

### How to Use a Drawer

`Drawer`s should be conditionally rendered.

`Drawer`s expose an `onCloseComplete` prop, this takes a callback which will be fired **after** exit animation has completed.

By default, they are pinned to the bottom of the screen, however, you can pin a `Drawer` to any side of the screen. Bottom pinned `Drawer`s are called `Tray`s and have a `HandleBar` which is used to drag the `Tray` on and off the screen.

`Drawer`s can render anything as children, however we recommend passing a callback as a child, and taking advantage of the exposed `handleClose` prop so children can toggle the visibility of the `Drawer`.

```jsx
function DefaultDrawer() {
  const [visible, { toggleOn, toggleOff }] = useToggler(false);
  return (
    <>
      <Button onPress={toggleOn}>Open</Button>
      {visible && (
        <Drawer pin={pin} onCloseComplete={toggleOff}>
          {({ handleClose }) => (
            <VStack spacing={2}>
              <LoremIpsum />
              <Button onPress={handleClose}>Close Drawer</Button>
            </VStack>
          )}
        </Drawer>
      )}
    </>
  );
}
```

### Overflow

`Drawer`s do not allow scrolling by default, to allow for pan gestures to work. However, you can enable scrolling by passing `disableCapturePanGestureToDismiss` and wrapping `children` in a `ScrollView`.

**NOTE**: If the children of a `ScrollView` are not interactable/pressable, you will need to wrap them in a `PressableOpacity` in order for scroll events to be captured by the `ScrollView`.

There is one caveat with this prop, since `disableCapturePanGestureToDismiss` stops event propagation from interactables within a `Drawer`, you lose the swipe to close functionality. But good news, is the `HandleBar` and `StatusBar` will still capture swipe to close gestures.

```jsx
function ScrollableDrawer() {
  const [visible, { toggleOn, toggleOff }] = useToggler(false);
  return (
    <>
      <Button onPress={toggleOn}>Open</Button>
      {visible && (
        <Drawer pin={pin} onCloseComplete={toggleOff} disableCapturePanGestureToDismiss>
          <VStack height={120} spacingHorizontal={3} spacingBottom={1}>
            <ScrollView>
              {/* Wrap non interactable content in a ScrollView with PressableOpacity */}
              <PressableOpacity noScaleOnPress>
                <LoremIpsum repeat={2} />
              </PressableOpacity>
            </ScrollView>
          </VStack>
          <Menu value={value} onChange={setValue}>
            <FlatList data={lotsOfOptions} renderItem={renderItem} />
          </Menu>
        </Drawer>
      )}
    </>
  );
}
```

A `Tray` is an elevated container that is pinned to the bottom of the screen. It also triggers an overlay. Its visibility is toggled by another component, typically a `Button` or a control, like a `Select`. It is the mobile equivalent of a `PopoverMenu`.

### How to Use a Tray

`Tray`s should be conditionally rendered, but only closed using the `handleClose()` method exposed on the Tray's `ref` or via a callback passed a `Tray`'s children; otherwise, if you conditionally close a Tray the exit animation won't work and lifecycle logic won't behave as intended.

By default, `Tray` is pinned to the bottom of the screen, if you want to pin it to a different side of the screen, use `Drawer` instead.

By default, `Tray`s have a `HandleBar` which is used to drag the `Tray` on and off the screen. It can be disabled with `hideHandleBar`.

### Render methods

#### Callback as children

`Tray`s can render anything as children, however we recommend passing a callback as a child, and taking advantage of the exposed `handleClose` prop so children can toggle the visibility of the `Tray`.

```jsx
function DefaultTray() {
  const [visible, { toggleOn, toggleOff }] = useToggler(false);
  return (
    <>
      <Button onPress={toggleOn}>Open</Button>
      {visible && (
        <Tray title="Example Title" onCloseComplete={toggleOff}>
          {({ handleClose }) => (
            <VStack spacing={2}>
              <LoremIpsum />
              <Button onPress={handleClose}>Close Tray</Button>
            </VStack>
          )}
        </Tray>
      )}
    </>
  );
}
```

#### Close with a ref

If you don't want to render children inside of a callback, you can also use the `handleClose` method exposed through a `ref` on the `Tray`.

```jsx
const TrayWithRef = () => {
  const [visible, { toggleOn, toggleOff }] = useToggler(false);
  const trayRef = useRef(undefined);

  const handleOptionPress = () => {
    trayRef.current?.handleClose();
  };

  return (
    <>
      <Button onPress={toggleOn}>Open</Button>
      {visible && (
        <Tray title={title} onCloseComplete={toggleOff} ref={trayRef}>
          <VStack spacing={2}>
            <LoremIpsum />
            <Button onPress={handleClose}>Close Tray</Button>
          </VStack>
        </Tray>
      )}
    </>
  );
};
```

### Overflow

`Tray`s do not allow scrolling by default, to allow for pan gestures to work. However, you can enable scrolling by passing `disableCapturePanGestureToDismiss` and wrapping `children` in a `ScrollView`.

**NOTE**: If the children of a `ScrollView` are not interactable/pressable, you will need to wrap them in a `PressableOpacity` in order for scroll events to be captured by the `ScrollView`.

There is one caveat with this prop, since `disableCapturePanGestureToDismiss` stops event propagation from interactables within a `Tray`, you lose the swipe to close functionality. But good news, is the `HandleBar` will still capture swipe to close gestures.

```jsx
function ScrollableTray() {
  const [visible, { toggleOn, toggleOff }] = useToggler(false);
  return (
    <>
      <Button onPress={toggleOn}>Open</Button>
      {visible && (
        <Tray pin={pin} onCloseComplete={toggleOff} disableCapturePanGestureToDismiss>
          <VStack height={120} spacingHorizontal={3} spacingBottom={1}>
            <ScrollView>
              {/* Wrap non interactable content in a ScrollView with PressableOpacity */}
              <PressableOpacity noScaleOnPress>
                <LoremIpsum repeat={2} />
              </PressableOpacity>
            </ScrollView>
          </VStack>
          <Menu value={value} onChange={setValue}>
            <FlatList data={lotsOfOptions} renderItem={renderItem} />
          </Menu>
        </Tray>
      )}
    </>
  );
}
```

### As a menu

If you want to render a list of `SelectOption`s as children of a `Tray`, you will need to wrap them in a `Menu` component. On mobile, this acts as a controlled input that keeps track of the currently selected value.

```jsx
const TrayMenu = () => {
  const [isTrayVisible, { toggleOff: handleClose, toggleOn: handleOpenTray }] = useToggler(false);
  const [value, setValue] = useState();
  const trayRef = useRef(undefined);

  const handleOptionPress = () => {
    trayRef.current?.handleClose();
  };

  return (
    <>
      <Button onPress={handleOpenTray}>Open</Button>
      {isTrayVisible && (
        <Tray title={title} onCloseComplete={handleClose} ref={trayRef}>
          {/* Make sure you pass both a value and onChange handler to Menu */}
          <Menu value={value} onChange={setValue}>
            {/* children can be anything, but in order to make it work like a select, you need to pass some SelectOptions */}
            {options.map((option: string) => (
              <SelectOption
                key={option}
                title={option}
                description="BTC"
                onPress={handleOptionPress}
                value={option}
              />
            ))}
          </Menu>
        </Tray>
      )}
    </>
  );
};
```

### Flows with Multiple Overlay Components

If you want an overlay component to trigger the visibility of another overlay component, you need to make sure the first component has completely dismounted before opening the next one. Overlay components expose lifecycle methods like `onDidClose` or `onCloseComplete` that can be used to trigger the next overlay component.

```jsx
function TrayToModalFlow() {
  const [isTrayVisible, { toggleOff: handleCloseTray, toggleOn: handleOpenTray }] =
    useToggler(false);
  const [isModalVisible, { toggleOff: handleCloseModal, toggleOn: handleOpenModal }] =
    useToggler(false);

  const handleTrayCloseComplete = useCallback(() => {
    handleCloseTray();
    handleOpenModal();
  }, [handleCloseTray, handleOpenModal]);

  return (
    <>
      <Button onPress={handleOpenTray}>Open Tray</Button>
      {isTrayVisible && (
        <Tray title={title} onCloseComplete={handleTrayCloseComplete}>
          {({ handleClose }) => {
            return (
              <>
                <HStack spacing={3}>
                  <LoremIpsum />
                </HStack>
                <HStack justifyContent="center">
                  <Button onPress={handleClose}>Open Modal</Button>
                </HStack>
              </>
            );
          }}
        </Tray>
      )}
      <Modal visible={isModalVisible} onRequestClose={handleCloseModal}>
        <ModalHeader title="I am a Modal" />
        <ModalBody>
          <LoremIpsum />
          <Button onPress={handleCloseModal}>Close</Button>
        </ModalBody>
      </Modal>
    </>
  );
}
```

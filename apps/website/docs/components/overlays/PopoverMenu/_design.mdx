### Usage

Use this component to display a list of options related to an existing UI element like an icon, button, action, or content.

### Types

There are a handful of ways to modify our Menu component. By default, all Popover Menus will be a simple list of items. If needed, you can adjust the style of the list item, add a description line, or place an object like an icon, avatar, or photo at the start of the list item.

### Content

Follow these content guidelines for popover menus.

**Concise and scannable**

- Inside the menu, use concise copy that’s easy for the user to scan.
- Avoid punctuation.

**Use appropriately**

- Avoid long lists of familiar data, such as a user’s birth year (which has 100+ options). Instead, let the user enter the info directly.
- Avoid using a popover menu for top level navigation or important items that should have higher visibility.

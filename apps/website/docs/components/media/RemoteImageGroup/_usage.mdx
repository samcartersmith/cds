### RemoteImageGroup

To size all the RemoteImage equally, pass the size prop.
To shape all the RemoteImage equally, pass the shape prop.

```jsx live
function Example() {
  const cryptoAssets = [
    assets.eth.imageUrl,
    assets.dai.imageUrl,
    assets.xrp.imageUrl,
    assets.ada.imageUrl,
  ];
  return (
    <ResponsiveExample>
      <RemoteImageGroup size="xxxl" shape="circle">
        {cryptoAssets.map((assetSrc) => (
          <RemoteImage source={assetSrc} borderColor="secondary" />
        ))}
      </RemoteImageGroup>
    </ResponsiveExample>
  );
}
```

### Adjust Size Using Avatar Sizes

You can adjust the size of the RemoteImage using any of the avatarSizes.

```jsx live
<ResponsiveExample>
  <RemoteImageGroup size="xxl" shape="circle">
    <RemoteImage source={assets.eth.imageUrl} borderColor="secondary" />
  </RemoteImageGroup>
</ResponsiveExample>
```

### Adjust Size Using Any Number

You can also adjust the size of the RemoteImage using any number. However,
we recommend that you use avatarSizes first. Only use arbitrary size if you really need it.

```jsx live
<ResponsiveExample>
  <RemoteImageGroup size={75} shape="circle">
    <RemoteImage source={assets.sushi.imageUrl} borderColor="secondary" />
  </RemoteImageGroup>
</ResponsiveExample>
```

### Max

To limit the amount of RemoteImages to show, use the max prop. It'll truncate the RemoteImage and show a "+X" label (where X is the remaining RemoteImage).
The size of the excess text ("+X") is relative to the size of RemoteImageGroup

```jsx live
function Example() {
  const faces = [squareAssets.human1, squareAssets.human2, squareAssets.human3];
  return (
    <ResponsiveExample>
      <VStack gap={2}>
        <RemoteImageGroup size={75} max={4}>
          {Object.values(squareAssets).map((face) => (
            <RemoteImage source={face} borderColor="secondary" />
          ))}
        </RemoteImageGroup>
        <RemoteImageGroup size="xl" max={4}>
          {Object.values(squareAssets).map((face) => (
            <RemoteImage source={face} borderColor="secondary" />
          ))}
        </RemoteImageGroup>
      </VStack>
    </ResponsiveExample>
  );
}
```

### Mix and Match

You can use different shapes in the children. Those shapes will take precedence over the shape prop of RemoteImageGroup.

```jsx live
function Example() {
  return (
    <ResponsiveExample>
      <RemoteImageGroup size="xxxl" shape="circle">
        <RemoteImage source={squareAssets.human1} shape="squircle" borderColor="secondary" />
        <RemoteImage source={assets.ada.imageUrl} borderColor="secondary" />
        <RemoteImage source={squareAssets.human3} shape="squircle" borderColor="secondary" />
        <RemoteImage source={assets.hbar.imageUrl} borderColor="secondary" />
      </RemoteImageGroup>
    </ResponsiveExample>
  );
}
```

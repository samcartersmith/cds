> [@cbhq/cds-(web|mobile)/layout/Spacer](https://github.cbhq.net/frontend/cds/blob/master/packages/web/layout/Spacer.tsx)

A specialized flexbox element that will automatically grow or shrink to fill available space, pushing sibling elements on their axis.

## Usage

The spacer is very situational, but is useful for pushing around siblings to ensure elements are in an ideal location (especially for mobile devices).

```jsx live
<VStack height={250}>
  <Box spacing={1} background>
    Top
  </Box>

  <Spacer />

  <Box spacing={1} background>
    Bottom
  </Box>
</VStack>
```

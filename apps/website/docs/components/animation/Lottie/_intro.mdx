### Color Override

To change the color of the animation in a mobile application, you can use the `colorFilters` property. First, identify the `nm` field within the lottie source file to determine the appropriate `keyPath` and the specific layer (found in the `layers` property) you want to modify. For instance, to change the colors in the `watchListStart` animation, refer to the structure outlined in the [lottie source file](https://github.cbhq.net/frontend/cds/blob/master/packages/lottie-files/src/watchListStar/watchListStar.json#L18).

Here's an example of how to implement this in JSX:

```jsx
<Lottie
  autoplay
  loop
  colorFilters={[
    { keypath: 'palette_primary', color: 'yellow' },
    { keypath: 'palette_foreground', color: 'orange' },
  ]}
  height={200}
  source={source}
  width={200}
/>
```

> [@cbhq/cds-web/system/MountComponent](https://github.cbhq.net/frontend/cds/blob/master/packages/web/system/MountComponent.ts)

`MountComponent` is a [Higher Order Component](https://reactjs.org/docs/higher-order-components.html) that checks if component has mounted and triggers a rerender so SSR UI's can hydrate properly (see [Perils of Rehydration](https://www.joshwcomeau.com/react/the-perils-of-rehydration/)). It can be used to prevent cumulative layout shifts (CLS) in server side rendered environments when content is dynamic.

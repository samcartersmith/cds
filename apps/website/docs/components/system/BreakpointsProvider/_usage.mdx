## Usage

You can instantiate your entire app or a single view to render as if it were at a specific device breakpoint.

```jsx
// Wrap the content you want to initially render as if it were at the `phone` breakpoint
// You can do this at the View or App level
<BreakpointsProvider device="phone">
  <ExampleStaticPage />
</BreakpointsProvider>;

// When this component first paints, it will use the `phone` breakpoint and display <MobileContent />
// If the client's browser is as wide as a desktop, it will rehydrate to render the <DesktopContent />
const ExampleStaticPage = () => {
  const { isPhone, isDesktop } = useBreakpoints();

  return isPhone ? <MobileContent /> : <DesktopContent />;
};
```

You can also use information from the server to figure out why type of device the client is using and conditionally pass the correct device to the `BreakpointsProvider` so that the first paint matches the user's device size.

```jsx
// Wrap your entire app
// get the user agent from the server
<BreakpointsProvider device={isMobileUserAgent ? 'phone' : undefined}>
  <App />
</BreakpointsProvider>;

// When this component first paints, it will use the `phone` breakpoint if the client's user agent
// matches that of a mobile device and display <MobileContent />
const ExampleStaticPage = () => {
  const { isPhone, isDesktop } = useBreakpoints();

  return isPhone ? <MobileContent /> : <DesktopContent />;
};
```

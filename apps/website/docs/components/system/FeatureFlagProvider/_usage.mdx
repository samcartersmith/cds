## Usage

:::caution

The direct child of `FeatureFlagProvider` _must be_ `ThemeProvider` in order for styles to load properly.

:::

### flexGap

There should be no visible change when toggling on `flexGap`. If you were to inspect with Dev Tools you should see Spacer DOM nodes are removed when `flexGap` is true.

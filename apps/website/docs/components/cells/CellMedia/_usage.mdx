`CellMedia` is a required component for displaying media within a cell and should only be used in that context. It automatically handles sizing and dimensions based on the current design system scale.

It supports the current media types:

- `icon` - An [Icon](/components/icon).
- `avatar` - A user [Avatar](/components/avatar) thumbnail/photo.
- `asset` - An asset logo.
- `image` - A standard [Image](/components/remote-image) (news articles, etc).
- `pictogram` - A [Pictogram](/components/pictogram).

When using an `icon`, an icon `name` is required.

```jsx live
<ExampleWithThemeToggles>
  <ListCell title="Send asset" media={<CellMedia type="icon" name="arrowUp" />} />
</ExampleWithThemeToggles>
```

When using the other types, an image `source` is required.

```jsx live
<ExampleWithThemeToggles>
  <ListCell
    title="Bitcoin"
    media={
      <CellMedia
        type="asset"
        source="https://dynamic-assets.coinbase.com/e785e0181f1a23a30d9476038d9be91e9f6c63959b538eabbc51a1abc8898940383291eede695c3b8dfaa1829a9b57f5a2d0a16b0523580346c6b8fab67af14b/asset_icons/b57ac673f06a4b0338a596817eb0a50ce16e2059f327dc117744449a47915cb2.png"
      />
    }
  />
</ExampleWithThemeToggles>
```

```jsx live
<ExampleWithThemeToggles>
  <ContentCell
    title="The Scoop"
    subtitle="The Block - Ethereum 2"
    media={
      <CellMedia
        type="image"
        source="https://dynamic-assets.coinbase.com/2954170d91149bea19e2d2eab8acc2f50ff4446b4b6fb09a7983ad7a481c636e6b29c5e09cf90f49f57dcef30ef7ff50bb99ad4fc068cf43265ad135c590fa7d/news_article_images/28216c10d33e3f2147fe05aa8a27bf4b9620dd658ce0be6c482b5629176e52e4.png"
      />
    }
  />
</ExampleWithThemeToggles>
```

When using a `pictogram`, providing a valid pictogram to the `illustration` prop is required.

:::note Pictogram props

`Pictogram` dimensions and scale are handled automatically, so setting the `dimensions` or `scaleMultiplier` props will have no effect.

<br />

`CellMedia` accessible labels (i.e. `accessibilityLabel` or `accessibilityHint`) take precedence over any accessible labels (i.e. `accessibilityLabel`, `accessibilityHint`, or `alt`) set on `Pictogram`.

:::

```jsx live
<ExampleWithThemeToggles>
  <ListCell
    title="Multi-factor authentication"
    media={<CellMedia type="pictogram" illustration={<Pictogram name="2fa" />} />}
  />
</ExampleWithThemeToggles>
```

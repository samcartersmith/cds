:::tip Accessibility tip

When a `ContentCell` is a pressable (i.e. when `onPress`, `onKeyPress`, `onKeyDown`, `to`, or `href` are defined), by default all of the visible text content in the cell will be read by assistive technology when it's focused. If you want to override this behavior on mobile, you can pass a custom label and hint to `accessibilityLabel` and `accessibilityHint` respectively. On web, you can set a custom label using `accessibilityLabelledBy` or `accessibilityLabel`. However, be mindful of any important content you may be hiding from the user.

<br />

**Web only**

When displaying a list of cells, wrap them in a `ul` and mark them as an `li`. These can both be accomplished with the `as` prop.

<br />

```jsx
<VStack as="ul">
  <ContentCell as="li" {...props} />
  <ContentCell as="li" {...props} />
  <ContentCell as="li" {...props} />
</VStack>
```

:::

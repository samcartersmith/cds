:::tip Accessibility tip

**Mobile only**

When a `ListCell` is a pressable (i.e. when a callback is set on `onPress`), by default all of the visible text content in the cell will be read by assistive technology when it's focused. If you want to override this behavior, you can pass a custom label and hint to `accessibilityLabel` and `accessibilityHint` respectively, but be mindful of any important content you may be hiding from the user.

<br />

**Web only**

When displaying a list of cells, wrap them in a `ul` and mark them as an `li`. These can both be accomplished with the `as` prop.

<br />

```jsx
<VStack as="ul">
  <ListCell as="li" {...props} />
  <ListCell as="li" {...props} />
  <ListCell as="li" {...props} />
</VStack>
```

:::

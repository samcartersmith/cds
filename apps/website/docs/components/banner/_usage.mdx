## Banner examples

These are a handful of common patterns you'll use when composing Banners with CDS. This list is not exhaustive, but should be a good starting point

```jsx live borderless
<Banner
  title="A compeling informational title (not exceeding two lines)"
  variant="informational"
  startIcon="info"
  primaryAction={<Link>Primary</Link>}
  secondaryAction={<Link>Secondary</Link>}
>
  Banner body content that can run over up to three lines maximum.
</Banner>
```

```jsx live borderless
<Banner
  title="A compeling informational title (not exceeding two lines)"
  variant="informational"
  startIcon="info"
  primaryAction={<Link>Primary</Link>}
  secondaryAction={<Link>Secondary</Link>}
  showDismiss
>
  Banner body content that can run over up to three lines maximum.
</Banner>
```

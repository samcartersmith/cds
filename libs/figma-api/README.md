# @cbhq/figma-api

REST API client for interacting with the Figma API.

## How to use

Import the `@cbhq/figma-api` package and call one of the `get` functions to make an [API request](https://www.figma.com/developers/api). Please see the JSDoc comments for each utility function for documentation for the endpoint.

All methods are strongly types using Figma's official type definitions, generated from their Open API specification.

## FAQ

- How do I provide Figma Access Token
  - Provide through a FIGMA_ACCESS_TOKEN env variable

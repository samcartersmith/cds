# @cbhq/icon-tasks

### Note on unicode character points

Our custom icon font is generated by mapping each unique icon SVG to a unicode code point in the unicode Private Use Area. We use the inclusive unicode range `0xF0000` - `0xFFFFD` (`983040` - `1048573`) in the Private Use Area of plane 15. This range includes 65534 unique unicode points, so there is a limit of 65534 icons per font. We avoid using the Basic Multilingual Plane to prevent our icons from falling back to emojis while loading on iOS.

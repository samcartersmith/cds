# Component Development Best Practices

Design Systems are by definition a system of disparate components. As a result they are inherently hard to keep in healthy place that allows them to scale well. To combat this challenge it is important that the whole team is on the same page about best practices. This page should serve as the jump off point for all things component development best practice.

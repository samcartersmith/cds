# Icons

There are 5 possible _dimensions_ for an icon **8px, 12px, 16px, 24px, and 32px**. When using
icons in applications that are wrapped in CDS ScaleProvider, use the relative _size_ enums **xs,
s, m, l, xl**. You can also set the color of icon. Colors are restricted to the color palette
which can be found <a href="https://cds.cbhq.net/foundation/color">here</a>.
On web, the icons are rendered as inline svgs, allowing you to pass valid svg html attributes to it.

## Icon library

Hover over an icon for 3s to see the icon name.

export const IconLookupTableStory = ({ data, color, iconSize, ...args }) => {
  const [query, setQuery] = useState('');
  const searchOnChange = throttle(event => {
    setQuery(event.target.value);
  }, 1000);
  return (
    <div {...args}>
      <div style={{ lineHeight: 2, height: 60 }}>
        <input
          style={{ border: '1px solid' }}
          onChange={searchOnChange}
          placeholder="Search Icon Here"
          type="text"
          id="search-icon"
          name="search-icon"
        />
      </div>
      {Object.keys(data)
        .filter(name => name.includes(query))
        .map(filteredName => {
          return (
            <div key={filteredName} title={filteredName} style={{ display: 'inline-block' }}>
              <Icon spacing={3} size={iconSize} name={filteredName} color={color} />
            </div>
          );
        })}
    </div>
  );
};

# Icons

There are 5 possible _dimensions_ for an icon **8px, 12px, 16px, 24px, and 32px**. There are also two illustration _styles_ **heavy** and **light**. Each kind of icon will offer some of those sizes and eventually both styles. When using icons in applications that are wrapped in CDS ScaleProvider, use the relative _size_ enums **xs, s, m, l, xl**. The icon components `IconHeavy` and `IconLight` will pick the correct pixel dimension icon asset for the current scale. On web, the icons are rendered as inline svgs, allowing you to pass valid svg html attributes to it.

<Source
  code={`
import { IconHeavy } from '@cb/design-system-icons'
import { ScaleProvider } from '@cb/design-system-web/primitives'\n
<ScaleProvider value="large">
  <IconHeavy kind="bell" size="m" title="notifications" titleId="ux-icon-story-bell-m" />
  <IconLight kind="heart" size="l" fill="red" title="like" titleId="ux-icon-story-like-l" />
</ScaleProvider>
`}
/>

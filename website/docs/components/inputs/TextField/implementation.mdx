### variant 

Determines the sentiment of the input. Because we allow startContent and endContent to be custom ReactNode, the content placed inside these slots will not change colors according to the variant. You will have to add that yourself

  mobileOptions={['foreground', 'foregroundMuted', 'negative', 'positive', 'primary']}

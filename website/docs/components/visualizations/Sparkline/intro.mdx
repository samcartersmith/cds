People with low vision often have difficulty viewing graphics that do not contrast with its background.
This can be exacerbated if the person has a color vision deficiency that lowers the contrast even further.
Providing a minimum luminance contrast ratio between the Sparkline stroke and its background can make the Sparkline more
accessible even if the person does not see the full range of colors. It also works for the rare individuals who see no color.

Under the hood, Sparkline uses `useAccessibleForegroundColor`, which does the following checks on a given color:

1. Check if color meets contrast criteria.
2. If pass, return color passed in.
3. If fail, check saturation.
4. If gray color return `gray100`.
5. If non gray color return closest CDS spectrum alias (i.e. `orange60`).

```jsx live
function Example() {
  const dimensions = { width: 64, height: 20 };
  const path = useSparklinePath({ ...dimensions, data: prices });
  return (
    <ExampleWithThemeToggles>
      <ListCell
        media={<CellMedia type="image" source={assets.btc.imageUrl} title="BTC" />}
        intermediary={<Sparkline {...dimensions} path={path} color={assets.btc.color} />}
        title={assets.btc.name}
        description={assets.btc.symbol}
        detail={prices[0]}
        subdetail="+4.55%"
        variant="positive"
        onPress={() => console.log('pressed')}
      />
    </ExampleWithThemeToggles>
  );
}
```

The `HStack` works exactly like `Box` but with children being automatically renderes as columns. Because of this, be sure that direct children are all block level elements.

```jsx live
<HStack>
  <Box spacing={1} background>
    First
  </Box>
  <Box spacing={1} background="backgroundAlternate">
    Second
  </Box>
  <Box spacing={1} background="backgroundOverlay">
    Third
  </Box>
</HStack>
```

Can provide a `gap` prop to inject spacing between each child.

```jsx live
<HStack gap={1}>
  <Box spacing={1} background>
    First
  </Box>
  <Box spacing={1} background="backgroundAlternate">
    Second
  </Box>
  <Box spacing={1} background="backgroundOverlay">
    Third
  </Box>
</HStack>
```

Icon buttons must have a name (which determines the icon), and can use variants to denote intent and importance.

```jsx live
<div>
  <IconButton name="orderHistory" onPress={console.log} variant="primary" />
  <IconButton name="gear" onPress={console.log} variant="secondary" />
  <IconButton name="checkmark" onPress={console.log} variant="foregroundMuted" />
  <IconButton name="orderHistory" onPress={console.log} variant="primary" transparent />
  <IconButton name="gear" onPress={console.log} variant="secondary" transparent />
  <IconButton name="checkmark" onPress={console.log} variant="foregroundMuted" transparent />
</div>
```

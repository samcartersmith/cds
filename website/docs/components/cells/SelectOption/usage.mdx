Supports the same props and usage as `ListCell`, but with styling specific to usage in a `Select` and `PopoverMenu`.

```jsx live
<SelectOption value="25" key="25" title="25" description="BTC" />
```

### Web

We strongly recommend that you use `SelectOption`s within `PopoverMenu`s and `Select`s. It has the same API as a `ListCell`, but with custom styles specific to usage within a `Select`.

#### Default Composition

```jsx live
<SelectOption value={option} title={option} description="BTC" />
```

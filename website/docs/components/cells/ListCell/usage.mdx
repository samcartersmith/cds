A cell row is divided into the following 4 columns:

- Start: A type of [media](./cell-media), like icon or avatar.
- Middle: The primary content; a title and description.
- End: Detailed information in neutral, negative, and positive variants.
- Accessory: A whitelisted icon displayed at the end.

```jsx live
<ListCell
  title="Bitcoin"
  description="BTC"
  detail="$45,123"
  subdetail="+4.55%"
  variant="positive"
  media={
    <CellMedia
      type="asset"
      source="https://dynamic-assets.coinbase.com/e785e0181f1a23a30d9476038d9be91e9f6c63959b538eabbc51a1abc8898940383291eede695c3b8dfaa1829a9b57f5a2d0a16b0523580346c6b8fab67af14b/asset_icons/b57ac673f06a4b0338a596817eb0a50ce16e2059f327dc117744449a47915cb2.png"
      title="BTC"
    />
  }
/>
```

Cells are also versatile enough to be used for pages like settings, where they can be used to drill down into sub-pages, or provide quick action controls.

```jsx live
<>
  <ListCell
    title="Security alerts"
    description="Push, email, SMS, in-app"
    accessory="arrow"
    onPress={console.log}
  />
  <ListCell
    title="Take a break"
    description="Pause notifications for a short time"
    action={<Switch checked onChange={() => {}} />}
  />
</>
```

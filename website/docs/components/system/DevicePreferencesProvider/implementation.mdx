## Props

### scale

Only use if you are passing in a value from a user preference since this will override device scale preference. You are also able to update the root scale imperatively via useRootScaleUpdater.

<ApiDetails
  componentName="DevicePreferencesProvider"
  propertyName="scale"
  required={false}
  defaultValue=""
  mobileOptions={['large', 'medium', 'small', 'system', 'xLarge', 'xSmall', 'xxLarge', 'xxxLarge']}
  webOptions={['large', 'medium', 'small', 'system', 'xLarge', 'xSmall', 'xxLarge', 'xxxLarge']}
/>

### spectrum

Pass in value from React Natives useColorScheme or some other user preference. You are also able to update the root spectrum imperatively via useRootSpectrumPreferenceUpdater.

Provides global spectrum and scale configuration for the app. This should be used at the root of the application.

If a scale is not provided in the prop it will attempt to determine the scale from the devices pixel ratio on mobile.

If a spectrum is not provided it will attempt to determine the default spectrum from the users operating system on web and mobile. It digs into the `(prefers-color-scheme: dark)` media query on the web to make this determination. For mobile it uses the result of the `useColorScheme()` react native hook.

The following hooks can be used in conjunction to the DevicePreferencesProvider:

`useRootScale` - Gets the currently configured root scale

`useRootScaleUpdater` - gets a function that can be used to update the root scale

`useRootSpectrum` - Gets the currently configured root spectrum

`useRootSpectrumUpdater` - gets a function that can be used to update the root spectrum

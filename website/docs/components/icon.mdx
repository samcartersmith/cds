### bordered

A boolean flag indicating whether or not a border should be shown around an icon. This border will match color prop. Border is only allowed for sizes m and above.

  mobileOptions={['false', 'true']}
  webOptions={['false', 'true']}

### title <Badge variant="web" />

Give the icon a description. It will show up in a tooltip after hovering over the icon for 3s. This is a default browser feature and very helpful to users to understand what the icon does.

<ApiDetails
  componentName="Icon"
  propertyName="title"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['string']}
/>

<ComponentDetails componentName="Icon" templateName="title" />

### titleId <Badge variant="web" />

If the icon is labelled by an independent element, then provide that elements id. This is used to set the aria-labelledby value on the icon svg. If the `title` prop is also provided, then `titleId` will be used to set the id on the `<title>` element.

<ApiDetails
  componentName="Icon"
  propertyName="titleId"
  required={false}
  defaultValue=""
  mobileOptions={[]}
  webOptions={['string']}
/>

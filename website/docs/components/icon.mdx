### size

Specifies the icon size. The actual icon size is determined based on this prop and the
current scale.

<ApiDetails
  componentName="Icon"
  propertyName="size"
  required={true}
  defaultValue=""
  mobileOptions={['l', 'm', 's', 'xs']}
  webOptions={['l', 'm', 's', 'xs']}
/>

<ComponentDetails componentName="Icon" templateName="size" />

### title

Give the icon a description. It will show up in a tooltip after hovering over the icon for 3s.
This is a default browser feature and very helpful to users to understand what the icon does.

<ApiDetails
  componentName="Icon"
  propertyName="title"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

<ComponentDetails componentName="Icon" templateName="title" />

### titleId

If the icon is labelled by an independent element, then provide that elements id. This is
used to set the aria-labelledby value on the icon svg. If the `title` prop is also provided,
then `titleId` will be used to set the id on the `<title>` element.

<ApiDetails
  componentName="Icon"
  propertyName="titleId"
  required={false}
  defaultValue=""
  mobileOptions={['string']}
  webOptions={['string']}
/>

<ComponentDetails componentName="Icon" templateName="titleId" />

Specifies text alignment. On mobile, the value `justify` is only supported on iOS and fallbacks to `start` on Android.

**[MDN Docs](https://developer.mozilla.org/en-US/docs/Web/CSS/text-align) | [React Native docs](https://reactnative.dev/docs/text-style-props#textalign)**

  componentName="PageTitle"
  propertyName="align"

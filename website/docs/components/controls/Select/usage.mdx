### Web Implementation

`Select` can take anything as a child, however, we strongly recommend that you use `SelectOption`. It has the same API as a `ListCell`, but with custom styles specific to usage within a `Select`. `SelectOption` also comes with a lot of baked in functionality, like keyboard navigation, accessibility properties, and focus behaviors.

#### Default Composition

```jsx live
function DefaultSelect() {
  const [value, setValue] = useState();
  const options = ['Option 1', 'Option 2', 'Option 3', 'Option 4', 'Option 5', 'Option 6'];
  return (
    <ExampleWithThemeToggles>
      <VStack spacing={2} background minHeight={400}>
        <Select value={value} placeholder="Choose something..." onChange={setValue}>
          {options.map((option) => (
            <SelectOption value={option} key={option} title={option} description="Description" />
          ))}
        </Select>
      </VStack>
    </ExampleWithThemeToggles>
  );
}
```

### Input Stack Options

The `Select` trigger can be customized similar to `TextInput`. These options are also available for mobile implementations.

#### Label and Helper Text

When space is tight, this brings the label inside of the Input. Should be used with a `compact` `SelectOption`.

```jsx live
function LabelHelperTextSelect() {
  const [value, setValue] = useState('Option 2');
  const options = ['Option 1', 'Option 2', 'Option 3', 'Option 4', 'Option 5', 'Option 6'];
  return (
    <ExampleWithThemeToggles>
      <VStack spacing={2} background minHeight={400}>
        <Select
          label="Make a selection"
          helperText="You can only choose one"
          value={value}
          onChange={setValue}
          startNode={<InputIcon name="calendar" />}
        >
          {options.map((option) => (
            <SelectOption value={option} key={option} title={option} description="Description" />
          ))}
        </Select>
      </VStack>
    </ExampleWithThemeToggles>
  );
}
```

#### Compact

When space is tight, this brings the label inside of the Input. Should be used with a `compact` `SelectOption`.

```jsx live
function CompactSelect() {
```

#### Variants

Variants can be used to surface positive or negative feedback.

```jsx live
function Variant() {
  const [value, setValue] = useState('Positive');
  const options = ['Positive', 'Negative', 'Primary', 'Secondary', 'Foreground'];
  return (
    <ExampleWithThemeToggles>
      <VStack spacing={2} background minHeight={400}>
        <Select
          variant={value.toLowerCase()}
          label="Make a selection"
          helperText="You can only choose one"
          value={value}
          onChange={setValue}
        >
          {options.map((option) => (
            <SelectOption value={option} key={option} title={option} />
          ))}
        </Select>
      </VStack>
    </ExampleWithThemeToggles>
  );
}
```

### Mobile Implementation <Badge variant="comingSoon" />

The mobile version of `Select` is quite different from web. When you interact with the trigger, it will toggle the visibility of a `Tray` which contains all the `SelectOption`s.

You can add a `title` as a prop to the`Tray`.

**Note**: `Tray` exposes a `handleClose` method as a prop to child functions. We recommend using the pattern shown below to control the visibility of the `Tray`.

#### Default Composition

```jsx
function DefaultSelectMobile() {
  const [isTrayVisible, { toggleOn, toggleOff }] = useToggler(false);
  const [value, setValue] = useState();
  const options = ['Option 1', 'Option 2', 'Option 3', 'Option 4', 'Option 5'];
  return (
    <Select value={value} onPress={toggleOn}>
      {isTrayVisible && (
        <Tray title={trayTitle} onCloseComplete={toggleOff}>
          {({ handleClose }) =>
            options.map((option) => {
              return (
                <SelectOption
                  title={option.label}
                  key={option.value}
                  onPress={handleClose}
                  onChange={setValue}
                  value={option}
                  selected={value === option.value}
                />
              );
            })
          }
        </Tray>
      )}
    </Select>
  );
}
```

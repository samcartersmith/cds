## Default 
The onChangeText is required for this to work. You must maintain your own
value state using useState, and pass the state to the value prop. Below is 
the most basic example of how to get this SearchInput working. 

```jsx live 
function SearchExample() {
	const [text, setText] = useState('');

	return (
		<>
			<SearchInput value={text} onChangeText={setText}  placeholder="Placeholder" />
			<TextLabel1 as="p">{text}</TextLabel1>
		</>
	);
}
```

## onSearch Example 
onSearch is triggered when user hits 'Enter' on both Mobile and Web. 
On Mobile, there are two ways to trigger onSearch. You can trigger by hitting 
'Enter' on Keyboard or by pressing the Search Icon when its available

```jsx live
function OnSearchExample() {
	const [text, setText] = useState('');

	const handleOnSearch = useCallback(
		(str) => {
			console.log(`User just typed: ${str}`);
		},
		[],
	)

	return (
		<>
			<SearchInput value={text} onChangeText={setText} onSearch={handleOnSearch} placeholder="Hit Enter" />
			<TextLabel1 as="p">{text}</TextLabel1>
		</>
	)
}
```

## onClear Example 
Add custom onClear logic example

```jsx live 
function OnClearExample() {
	const [text, setText] = useState('');

	return (
		<SearchInput 
			onClear={() => console.log('Clearing...')} 
			onChangeText={setText}
			value={text}
			placeholder="Placeholder" 
		/>
	)
}
```

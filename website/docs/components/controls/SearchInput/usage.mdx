## Clearing Logic 
Try typing something in the input. The close IconButton should appear when there is some value. When 
value is empty again, it should disappear. When you click on the clear 
button it should clear the text AND log Clearing....

```jsx live 
function ClearButtonAppears() {
	return (
		<SearchInput 
			onClear={() => console.log('Clearing...')} 
			placeholder="Placeholder" 
		/>
	)
}
```

## Props

### align

Specifies text alignment. On mobile, the value `justify` is only supported on iOS and fallbacks to left on Android.

**[MDN Docs](https://developer.mozilla.org/en-US/docs/Web/CSS/text-align) | [React Native docs](https://reactnative.dev/docs/text-style-props#textalign)**

<ApiDetails
  componentName="Text"
  propertyName="align"
  required={false}
  defaultValue="start"
  mobileOptions={['center', 'end', 'justify', 'start']}
  webOptions={['center', 'end', 'justify', 'start']}
/>

<ComponentDetails componentName="Text" templateName="align" />

**[MDN Docs](https://developer.mozilla.org/en-US/docs/Web/CSS/user-select) | [React Native Docs](https://reactnative.dev/docs/text#selectable)**

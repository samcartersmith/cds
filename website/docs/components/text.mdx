## Props

### align

Specifies text alignment. On mobile, the value `justify` is only supported on iOS and fallbacks to left on Android.

**[MDN Docs](https://developer.mozilla.org/en-US/docs/Web/CSS/text-align) | [React Native docs](https://reactnative.dev/docs/text-style-props#textalign)**

<ApiDetails
  componentName="Text"
  propertyName="align"
  required={false}
  defaultValue="left"
  mobileOptions={['center', 'left', 'right', 'start', 'end', 'justify']}
  webOptions={['center', 'left', 'right', 'start', 'end', 'justify']}
/>

<ComponentDetails componentName="Text" templateName="align" />

### selectable

Set select behavior. On mobile, selectable is binary, none will be not selectable and either text or all will be selectable on Android. It is not yet supported on iOS.

**[MDN Docs](https://developer.mozilla.org/en-US/docs/Web/CSS/user-select) | [React Native Docs](https://reactnative.dev/docs/text#selectable)**

<ApiDetails
  componentName="Text"
  propertyName="selectable"
  required={false}
  defaultValue="text"
  mobileOptions={['none', 'text', 'all']}
  webOptions={['text', 'none', 'all']}
/>

<ComponentDetails componentName="Text" templateName="selectable" />

### Style changes

<Image light="/img/frontier/typestyles.png" dark="/img/frontier/typestyles_dark.png" />

To allow teams more flexibility in establishing hierarchy especially at the larger text sizes, we’ll be introducing Display 3 and Title 4.

Display 2’s text size has also been increased to better fit the new type ramp. This will be available behind a FrontierTypography feature flag. If you’re using the current Display2, we recommend starting to migrate to other type styles.

### Mono

<Image light="/img/frontier/mono.png" dark="/img/frontier/mono_dark.png" />

We’ve heard from teams that there’s a common need to show users wallet address and code snippets, so we’ve added Coinbase Mono to our type offerings. Now, you can set any text component to Mono, and the typeface will adjust.

### Figma Text component

<Image src="/img/frontier/figma_text.png" />

We’ve added a Text component to Figma. We’ve heard you about creating links, setting tabular numbers, and now with being able to set Mono we think it’s time to more easily set that. The Text component will better model the capabilities of CDS Text available in code.

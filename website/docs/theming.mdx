### Scale

**xSmall | small | medium | large | xLarge | xxLarge | xxxLarge**

The `scale` prop in `ThemeProvider` determines which sizes to pull for typography, spacing, icons and layout and is based on an applications information density.

```jsx {5}
function MyApp() {
  import { ThemeProvider } from '@cds/theme';
  return (
    // xSmall | small | medium | large | xLarge | xxLarge | xxxLarge
    <ThemeProvider scale="small">
      <AppContent />
    </ThemeProvider>
  );
}
```

### Spectrum

**light | dark**

The `spectrum` prop in ThemeProvider determines which spectrum a palette will pull from. For example, in the light spectrum, gray100, maps to black while in the dark spectrum gray100 maps to white. The default palette CDS ships with assigns foreground to gray100. `foreground` is then used in majority of our text components and is dark mode compatible by default.

```jsx {4}
function MyApp() {
  return (
    // light | dark
    <ThemeProvider spectrum="dark">
      <AppContent />
    </ThemeProvider>
  );
}
```

The spectrum is comprised of the following colors:

- **Hues**: blue | green | orange | yellow | gray | indigo | pink | purple | red
- **Hue Steps**: 0 | 5 | 10 | 15 | 20 | 30 | 40 | 50 | 60 | 70 | 80 | 90 | 100

### Palette

CDS comes with the default palette below. If a portion of a palette is provided it will be merged with the closest parent ThemeProvider if present, if not it will be fallback to merging with the default CDS palette.

```jsx {28}
import { ThemeProvider } from '@cds/theme';
/**
 * Key is PaletteAlias i.e. foreground
 * Value is SpectrumAlias or SpectrumAliasWithOpacity i.e. gray100 or [gray100, 0.5].
 */
export const myPalette = {
  foreground: 'gray100',
  foregroundMuted: 'gray60',
  background: 'gray0',
  backgroundAlternate: 'gray5',
  backgroundOverlay: ['blue90', 0.33],
  divider: ['gray60', 0.33],
  stroke: ['gray60', 0.66],
  primary: 'blue60',
  primaryForeground: 'gray0',
  negative: 'red60',
  negativeForeground: 'gray0',
  positive: 'green60',
  positiveForeground: 'gray0',
  secondary: 'gray0',
  secondaryForeground: 'gray100',
};

function MyApp() {
  return (
    <ThemeProvider palette={myPalette}>
      <AppContent />
    </ThemeProvider>
  );
}
```

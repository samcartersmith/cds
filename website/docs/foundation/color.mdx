CDS color primitive aligns with the Coinbase Brand refresh work and makes it easy to create accessible designs that can be light or dark. Our color system is comprised of Spectrum, which is a robust color library, and Palettes, which is a subset of Spectrum geared towards aligning common usages across products.

## Spectrum

Spectrum is an extensive color library with 8 color sets that has both light and dark mode versions.

- Spectrum was based off the Brand teams color work.
- Every color used in an interface should come from Spectrum and pulled into a palette.
- All color sets are generated by our algorithm that ensures a consistent color breakdown across the various color spaces.

All of the CDS spectrum colors will auto-invert in dark mode. For example, gray100 in light mode is dark gray in light mode and will become white in dark mode, which ensures that dark text/icon colors in Light Mode will automatically become light in Dark mode.

Similarly, colors in light mode will invert automatically to a darker shade in dark mode that appears more muted, which helps prevent a vibration in the eye.

:::note
It's a good idea to visually QA interfaces in Dark Mode to ensure the color usage and adaption is what you intend.
:::

## Palette

Palettes aim to align common color usages across interfaces. All of the colors used in our palettes come from spectrum colors. Using palette colors instead of directly accessing spectrum colors ensure:

- Accessibility & consistency
- A seamless switch between light and dark modes
- Easily themeable by an alternative brand

### Configuration

```jsx live
<ThemeProvider
// Uncomment to override spectrum manually or toggle dark mode in nav to see changes
// spectrum="light"
>
  <div>
    <TextDisplay1 as="h1" color="primary">
      Display1
    </TextDisplay1>
    <ThemeProvider palette={{ primary: 'pink60' }}>
      <TextDisplay1 as="h1" color="primary">
        Display1
      </TextDisplay1>
    </ThemeProvider>
  </div>
</ThemeProvider>
```

## Figma

### Integration

All CDS Colors are available in Figma through four Libraries:

- CDS Dense 🌞
- CDS Dense 🌙
- CDS Normal 🌞
- CDS Normal 🌙

All libraries provide on instance of CDS's color system, so you'll have access to Spectrum and Palettes.

- If you're designing for Light Mode use a library with this symbol 🌞
- If you're designing for Dark Mode use a library with this symbol 🌙

:::note
Do not mix up color library usages. You only may use 🌞 or 🌙 one at a time.
:::

### Configuration

You may adjust the config to change the Spectrum colors in the Palette. This will allow you greatly change the look of an interface.

## Handoff

You can easily access the spectrum through the Figma color picker and CDS Libraries

- Please opt to use palette colors, like “Foreground Primary”, instead of spectrum colors, like “Gray 100”, when they are covered.
- Design QA any direct uses of spectrum for visual aesthetics and accessibility.

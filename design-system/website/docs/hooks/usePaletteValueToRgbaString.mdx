> [@cbhq/cds-(web|mobile)/color/usePaletteValueToRgbaString](https://github.cbhq.net/mono/repo/blob/master/eng/shared/design-system/web/color/usePaletteValueToRgbaString.ts)

Convert a `PaletteValue` (`blue60` or `[blue60, 0.4]`) to an rgba string.

:::danger
This is intented to only be used internally.
Only use this if you are using an integration which does not support CSS variables.
Please reach out in #ask-cds for other use cases that might require leveraging this.
:::

```jsx live
function Example() {
  const paletteConfig = usePaletteConfig();
  const primaryRgbaString = usePaletteValueToRgbaString(paletteConfig.primary);
  return (
    <ExampleWithThemeToggles>
      <HStack gap={1} alignItems="center">
        <ColorTile size={15} color={primaryRgbaString} />
        <TextLabel2 as="span">{primaryRgbaString}</TextLabel2>
      </HStack>
    </ExampleWithThemeToggles>
  );
}
```

If hook does is used without any arguments it returns a function to call imperatively.

```jsx live
function Example() {
  const paletteConfig = usePaletteConfig();
  const toRgbaString = usePaletteValueToRgbaString();

  return (
    <ExampleWithThemeToggles>
      <VStack gap={1}>
        {Object.entries(paletteConfig).map(([name, value]) => {
          const rgbaString = toRgbaString(value);
          return (
            <HStack gap={1} alignItems="center">
              <TextLabel1 as="span">{`${name}: `}</TextLabel1>
              <ColorTile size={15} color={rgbaString} />
              <TextLabel2 as="span">{rgbaString}</TextLabel2>
            </HStack>
          );
        })}
      </VStack>
    </ExampleWithThemeToggles>
  );
}
```

> [@cbhq/cds-common/palette/usePaletteConfig](https://github.cbhq.net/mono/repo/blob/master/eng/shared/design-system/common/palette/usePaletteConfig.ts)

`usePaletteConfig` returns the palette config from the closest parent `<ThemeProvider />`.

- The `gray100` or `['gray80', 0.33]` are the two types of `PaletteValue` that CDS supports.
- Any utility function that is prefixed with `paletteValueTo` will take a `PaletteValue` and convert it to some other format.

```jsx live
function Example() {
  const paletteConfig = usePaletteConfig();
  return (
    <VStack gap={1}>
      {Object.entries(paletteConfig).map(([key, value]) => (
        <HStack gap={1}>
          <TextLabel1 as="span">{`${key}:`}</TextLabel1>
          <TextLabel2 as="span">
            <code>{JSON.stringify(value)}</code>
          </TextLabel2>
        </HStack>
      ))}
    </VStack>
  );
}
```

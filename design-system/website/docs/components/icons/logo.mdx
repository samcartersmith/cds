## Mark

The mark comes in 2 sizes 16 and 32 and 2 variants light and dark, which correspond to whichever spectrum is being displayed. The mark cannot go below 16px, but beyond 32px it’ll scale up indefinitely.

```jsx live
<HStack>
  <Box spacingEnd={2}>
    <LogoMark size={16} />
  </Box>
  <Box spacingEnd={2}>
    <LogoMark size={32} />
  </Box>
</HStack>
```

## Wordmark

The wordmark comes in 3 color variants. By default it will be the brand blue. If `foreground` prop is present it will use a black color if spectrum is `light` and a white color if spectrum is `dark`. Minimum height is 16px tall and no maximum.

```jsx live
<VStack>
  <Box width={200} spacingBottom={2}>
    <LogoWordmark />
  </Box>
  <ThemeProvider spectrum="light">
    <Box background width={200} spacingBottom={2}>
      <LogoWordmark foreground />
    </Box>
  </ThemeProvider>
  <ThemeProvider spectrum="dark">
    <Box background width={200}>
      <LogoWordmark foreground />
    </Box>
  </ThemeProvider>
</VStack>
```

## Favicon

There’s 2 sizes of favicons (1 if we want to go with svg) with light and dark variants. The light and dark look the same currently, but that allows us to provide different assets later on.

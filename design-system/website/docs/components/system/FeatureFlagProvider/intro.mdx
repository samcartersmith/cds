### What is the process for rolling out a new feature flag?

- Add feature flag to CDS library and cut a new release
- Announce feature flag is available in #announcements-cds
- Provide a launch date for moving from beta to stable
- Product testing/feedback period
- Product pillar sign-off on launch date
- Launch date
  - Toggle feature flag default from false to true (no longer opt-in)
  - Cut a new release
  - Post in #announcements-cds that feature has moved to stable
- Deprecate feature flag
  - Cleanup conditional code

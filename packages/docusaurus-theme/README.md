# docusaurus-theme

The docusaurus-theme package contains the default theme for CDS documentation sites built with Docusaurus.

## Commands

| Script           | Description                   |
| ---------------- | ----------------------------- |
| `yarn build`     | Build the theme               |
| `yarn watch`     | Watch for changes and rebuild |
| `yarn format`    | Format code                   |
| `yarn lint`      | Lint code                     |
| `yarn test`      | Run tests                     |
| `yarn typecheck` | Type check code               |

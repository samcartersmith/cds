# Product Module: Base (TBA)

> Apply these rules ONLY when the designer specifies Base / TBA as the product area.
> These rules OVERRIDE the general guidelines where noted.

## Capitalization — OVERRIDES general sentence case

Base uses **Title Case** for:

- Page titles, section headers, alert/modal titles
- Buttons, navigation elements (tabs, menus)

Title Case rules:

- Capitalize all words EXCEPT: a, an, the, and, but, or, for, at, by, in, of, on, to, up, with
- ALWAYS capitalize the first and last word
- ALWAYS capitalize all parts of hyphenated compounds: "A Step-By-Step Guide"

Base uses **sentence case** for:

- Body copy, subtitles, hint text, component labels, links, error/success message bodies

## Tone

- "Punchy clarity" — brief and direct
- Lively, fun host of the onchain world
- Optimistic realists — acknowledge where tech is today
- Playful, bold, attitudinal, friendly, welcoming, human
- "Speak in words, not acronyms"
- Never aggressive, demanding, arrogant, or patronizing

# Stories folder

The stories folder stores functions that can codegenerates stories that have dynamic data.

This is necessary because Storybook v7 onwards is no longer supporting storiesOf feature. Hence, stories like Icons and Illustrations can no longer use this feature to chunk the stories up.

More information about storiesOf being dropped from Storybook v7 [here](https://github.com/storybookjs/storybook/blob/next/MIGRATION.md#individual-story-decorators)

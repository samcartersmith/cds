# Text

`Text` is a component that allows us to show text in different typographies. Make sure that your app is wrapped with a `ScaleProvider` before using one of the typography text components. The style also depends on which scale is app is using. See [design documentation](https://docs.google.com/document/d/1-pl0GcLzJ3RAnkh3A5s6VpvVwNnndPcptlKjMj_3jRQ/edit).

For accessibility purpose, heading tags, i.e. `h1`-`h6`, imply hierarchical information and thus it is important for heading tags to be applied for heading typographies. For non-heading typographies like body, label, and legal, you can use any html tags for text that makes the most sense for you by using `as` prop.

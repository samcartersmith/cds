# Text

`Text` is a component for showing text in different typography. The typography sizes vary based on the scale of the closest ancestor that has a scale class applied. If no scale class is applied to the entire document, the text components will not have the appropriate typography styles because styles are defined by CSS variables and the scale classes define the values for the CSS variables. See [design documentation](https://docs.google.com/document/d/1-pl0GcLzJ3RAnkh3A5s6VpvVwNnndPcptlKjMj_3jRQ/edit).

For accessibility purposes, heading tags, i.e. `h1`-`h6`, carry hierarchy information and therefore it is important for heading tags to be used consistently. They also matter for SEO. CDS enforces that h1-h5 are always associated with the same typography styles. Optionally, any typography can be used with any non-heading html text tag of choice.

## Usage

<Source
  code={`
import { TextDisplay2, TextBody } from '@cb/design-system-web/components'
import { ScaleProvider } from '@cb/design-system-web/primitives'
\n
// Make sure the scale class name is applied somewhere in the ancestor tree.
// Only once in the app is needed unless the app is using multiple scales.
<ScaleProvider value="large">
  <TextDisplay2 color="#161652" textAlign="center">Coinbase Design System</TextDisplay2>
  {/* Title2 is h4 by default */}
  <TextTitle2 as="span">Earn BTC</TextTitle2>
  <TextBody as="code" display="inline" tnum>2020</TextBody>
  <TextLegal wrap as="p">Buy and sell popular digital currencies</TextLegal>
<ScaleProvider>
`}
/>

# Coinbase Design System

Some important links you should definitely know about:

- [Contributing to the CDS Codebase](./CONTRIBUTING.md) - This is great for setting up your CDS dev environment and learning more about what your dev cycles will look like.
- [The CDS Website](https://cds.cbhq.net/) - This is where we document CDS best practices, principles, components and more for other teams to consume. It is our primary communication channel to CDS consumers.
- [CDS Confluence Home](https://confluence.coinbase-corp.com/pages/viewpage.action?spaceKey=CDS&title=CDS+Home) - If you aren't finding what you need in our github documentation this is a great place to look.
